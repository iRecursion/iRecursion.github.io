1. Model Classify

  • 审核端点(moderation endpoint)：
    可用于检查文本是否具有潜在危害potentially harmful；

  • 工程师可使用它来识别可能有害的内容并采取行动，如过滤；

  • 大多工程师可免费使用moderation endpoint，为获得更高的准确度accuracy，
    请尝试将长文本分割成较小的块，每个块少于2000个字符；

  • splitting long text piece into
    smaller chunk each less than 2000 character；

1.1. hate

  • content that expresse，incite，promote hate
    based on race，gender，ethnicity，religion，
    nationality，sexual orientation，disability status，caste；

  • 表达、煽动或宣扬基于种族、性别、民族、宗教、国籍、性取向、残疾状况或种姓的仇恨的内容；

  • 针对非受保护群体(如棋手)的仇恨内容属于骚扰，
    hateful content aimed at
    non-protected group(chess player) is harassment；

1.2. hate/threatening

  • 仇恨内容还包括基于种族、性别、民族、宗教、国籍、性取向、
    残疾状况或种姓对目标群体实施暴力或严重伤害；

  • hateful content that also include violence or
    serious harm toward target group based on race，gender，ethnicity，
    religion，nationality，sexual orientation，disability status，caste；

1.3. harassment

  • content that expresse，incite，
    promote harassing language toward any target；

  • 对任何目标表达、煽动或宣扬骚扰性语言的内容；

1.4. harassment/threatening

  • 骚扰内容还包括对任何目标的暴力或严重伤害；

  • harassment content that also include
    violence or serious harm toward any target；

1.5. self-harm

  • content that promote，encourage，depict act of self-harm，
    such as suicide，cutting，eating disorder；

  • 宣扬、鼓励或描述自残行为的内容，如自杀、自残和饮食失调；

1.6. self-harm/intent

  • 说话者表示他们正在或打算进行自残行为的内容，例如自杀、割伤和饮食失调；

  • content where speaker expresse that they are engaging or intend to
    engage in act of self-harm，such as suicide，cutting，eating disorder；

1.7. self-harm/instructions

  • content that encourage performing act of self-harm，
    such as suicide，cutting， eating disorder，
    or that give instruction or advice on how to commit such act；

  • 鼓励进行自残行为：(如自杀、割伤和饮食失调)的内容，
    或提供有关如何实施此类行为的说明或建议的内容；

1.8. sexual

  • 旨在引起性兴奋的内容，如性活动的描述，或宣传性服务(不包括性教育和健康)；

  • content meant to arouse sexual excitement，
    such as description of sexual activity，
    or promote sexual service(excluding sex education and wellness)；

1.9. sexual/minors

  • 涉及18岁以下个人的色情内容；

  • sexual content include individual who under 18 year old；

1.10. violence

  • 描述死亡、暴力或身体伤害的内容；

  • content that depict death，violence，physical injury；

1.11. violence/graphic

  • 以图形细节描述死亡、暴力或身体伤害的内容；

  • content depict death，violence，physical injury in graphic detail；

2. Quick Start

  • 要获取一段文本的分类，请向审核端点发出请求，
    to obtain classification for text piece；

curl https://api.openai.com/v1/moderations \
	-X POST \
  	-H "Content-Type: application/json" \
  	-H "Authorization: Bearer $OPENAI_API_KEY" \
  	-d '{"input": "sample text"}'

2.1. flagged

  • 若模型将内容分类为潜在有害则设置为true，否则设置为false；

  • set to true if model classify content
    as potentially harmful，false otherwise；

2.2. categories

  • 包含每个类别违规标志的字典；

  • 对每个类别，若模型将相应类别标记为违规，则值为true，否则为false；

  • contain dictionary of per-category violation flag，for each category；

  • true if model flag corresponding category as violate，false otherwise；

2.3. category_scores

  • 包含模型输出的每个类别原始分数的字典，表示模型对输入违反OpenAI该类别策略的置信度；

  • 值范围：0 ~ 1，值越高即置信度越高，分数不应被解释为概率；

  • contain dictionary of per-category raw score output by model；

  • denoting model confidence input violate OpenAI policy for category；

  • value scope：0 ~ 1，higher value denote higher confidence，
    score should not be interprete as probability；

{

    "model": "text-moderation-007",

            "categories": {
