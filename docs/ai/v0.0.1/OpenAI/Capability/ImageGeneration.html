1. Intro

  • 用DallE生成或操作manipulate图像；

  • A：根据文本提示(text prompt)从头开始创建图像，DallE3和DallE2；

  • B：根据新的文本提示，通过让模型替换预先存在的图像的某些区域来创建图像的编辑版本，仅限DallE2；

  • C：创建现有图像的变体variation，仅限DallE2；

  • 使用DallE3：https://chatgpt.com；

  • 使用DallE2：https://labs.openai.com；

2. Usage

2.1. Generation

  • 图像生成端点允许根据文本提示创建原始图像，用DallE3时，
    图像的尺寸可为1024 * 1024、1024 * 1792或1792 * 1024像素；

  • 默认：图像以标准质量生成，但使用DallE3时，可设置质量：
    "hd"以增强细节；方形square，标准质量的图像生成速度最快；

  • 可使用DallE3一次请求一个图像(通过发送并行请求更多图像)，
    或带有n参数的DallE2一次最多请求10个图像；

from openai import OpenAI
client = OpenAI()

response = client.images.generate(
  	model="dall-e-3",
  	prompt="white siamese cat",
  	size="1024x1024",
  	quality="standard",
  	n=1,
)
image_url = response.data[0].url

2.2. Prompting

  • DallE3模型采用提供的默认提示，并出于安全原因自动重写它，并添加更多细节；

  • 虽目前无法禁用此功能，但可通过在提示中添加以下内容来使用提示来使输出更接近所请求的图像：
    需测试该工具如何使用极其简单的提示来工作，不要添加任何细节，只需按原样使用即可：

  • 更新后的提示在数据响应对象的revised_prompt字段中可见；

  • 使用response_format参数，每个图像都可作为URL或Base64数据返回，URL将在一小时后过期；

2.3. Edit DallE2 Only)

  • 图像编辑端点也称修复(inpainting)，允许通过上传图像和指示应
    替换那些区域的mask indicating来编辑或扩展(extend)图像；

  • mask indicate的透明区域transparent area指示应编辑图像的位置，
    且提示应描述完整的图像，而不仅仅是擦除的区域erased area；

from openai import OpenAI
client = OpenAI()

response = client.images.edit((
  	model="dall-e-2",
  	image=open("sunlit_lounge.png", "rb"),
  	mask=open("mask.png", "rb"),
  	prompt="A sunlit indoor lounge area with a pool containing a flamingo",
  	n=1,
  	size="1024x1024"
)
image_url = response.data[0].url

  • 上传的图像和mask必须都是大小小于4MB的方形png图像，且尺寸(dimension)必须相同，
    生成输出时不会使用遮罩的非透明区域，故它们不一定需像上面的案例一样与原始图像匹配；

2.4. Variation DallE2 Only

  • 图像变体端点允许生成给定图像的变体；

  • 类似于编辑端点，输入图像必须是png格式，且小于4MB；

from openai import OpenAI
client = OpenAI()

response = client.images.create_variation(
  	model="dall-e-2",
  	image=open("corgi_and_cat_paw.png", "rb"),
  	n=1,
  	size="1024x1024"
)

image_url = response.data[0].url

2.5. Content Moderation

  • 提示和图像会基于content policy过滤，若提示或图像被标记(flag)将返回错误；

3. Language Specific Tip

3.1. Via In Memory Img Data

  • 一般使用open函数从磁盘读取图像数据，某些情况下可能会将图像数据存储在内存中，
    使用存储在BytesIO对象中的图像数据的Api调用：

from io import BytesIO
from openai import OpenAI
client = OpenAI()

# BytesIO object that contain image data
byte_stream: BytesIO = [your image data]
byte_array = byte_stream.getvalue()
response = client.images.create_variation(
  	image=byte_array,
  	n=1,
  	model="dall-e-2",
  	size="1024x1024"
)

3.2. Operating on Img Data

  • 将图像传递给Api；

from io import BytesIO
from PIL import Image
from openai import OpenAI
client = OpenAI()

# read image from disk and resize
image = Image.open("image.png")
width, height = 256, 256
image = image.resize((width, height))

# convert image to BytesIO object
byte_stream = BytesIO()
image.save(byte_stream, format='PNG')
byte_array = byte_stream.getvalue()

response = client.images.create_variation(
  	image=byte_array,
  	n=1,
  	model="dall-e-2",
  	size="1024x1024"
)

3.3. Error Handling

  • 因输入无效，速率限制或其它问题，api请求可能返回错误，error可通过
    try…​ except语句来处理，错误详细信息可在e.error中找到：

import openai
from openai import OpenAI
client = OpenAI()

try:
  	response = client.images.create_variation(
    	image=open("image_edit_mask.png", "rb"),
    	n=1,
    	model="dall-e-2",
    	size="1024x1024"
  	)
  	print(response.data[0].url)
except openai.OpenAIError as e:
  	print(e.http_status)
  	print(e.error)
