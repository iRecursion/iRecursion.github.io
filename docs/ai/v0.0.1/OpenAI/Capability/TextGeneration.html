1. Chat Completion

1.1. Text Generation Model

  • 文本生成模型(通常称为生成式预训练Transformer或大型语言模型)经过训练可理解
    自然语言、代码和图像，这些模型提供文本输出来响应其输入，这些模型的输入也称为提示prompt；

  • 设计提示本质上是如何"编程program"大型语言模型，
    通常是通过提供instruction或一些如何成功完成任务的示例；

  • 随gpt-4-turbo的发布，可构建还可处理和理解图像的系统；

  • 使用文本生成模型，可构建应用程序来：

    • draft document

    • write computer code

    • answer question about knowledge base

    • analyze text

    • give software natural language interface

    • tutor in subject range

    • translate language

    • simulate game character

Model Family Endpoint

Newer Model(2023–)

gpt-4，gpt-4-turbo-preview，gpt-3.5-turbo

https://api.openai.com/v1/chat/completions

Updated Legacy Model(2023)

gpt-3.5-turbo-instruct，babbage-002，davinci-002

https://api.openai.com/v1/completions

1.2. Chat Completion Api

  • Chat Model将消息列表作为输入，并返回模型生成的消息作为输出；

  • 尽管chat format旨在使多轮对话变得容易，但它对没任何对话的单轮任务也同样有用；

from openai import OpenAI
client = OpenAI()

response = client.chat.completions.create(
  	model="gpt-3.5-turbo",
  	messages=[
	    {"role": "system", "content": "You are a helpful assistant."},
	    {"role": "user", "content": "Who won the world series in 2020?"},
	    {"role": "assistant", "content": "The Los Angeles Dodgers won the World Series in 2020."},
	    {"role": "user", "content": "Where was it played?"}
  	]
)
