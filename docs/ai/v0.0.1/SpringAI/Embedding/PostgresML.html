1. Intro

  • Spring AI支持PostgresML text embeddings model；
    Embedding是文本的数字表示，它们被用来将单词word和句子sentence表示为向量，数字数组；

  • Embedding可通过用距离测量distance measure比较数字向量
    numeric vector的相似性similarity来找到相似的文本片段piece，
    也可用作其他ML模型的输入特征，因大多数算法不能直接使用文本；

  • 许多预训练的LLM可用于从PostgresML中的文本生成embeddings，
    我们可浏览所有可用的模型，在Hugging Face找到的最佳解决方案；

<dependency>
	<groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-postgresml-spring-boot-starter</artifactId>
</dependency>

2. Embedding Properties

  • 用spring.ai.postgresml.embedding.options.*
    属性来配置PostgresMlEmbeddingClient

Property Description Default Value

spring.ai.postgresml.embedding.enabled

启用PostgresML embedding client

true

spring.ai.postgresml.embedding.options.transformer

用于embedding的Huggingface transformer model

distilbert-base-uncased

spring.ai.postgresml.embedding.options.kwargs

additional transformer specific option

空map

spring.ai.postgresml.embedding.options.vectorType

用作embedding的PostgresML向量类型：PG_ARRAY或PG_VECTOR

PG_ARRAY

spring.ai.postgresml.embedding.options.metadataMode

Document metadata aggregation mode

EMBED

3. EmbeddingOptions

  • PostgresMlEmbeddingOptions.java

  • 用PostgresMlEmbeddingOptions.java为
    PostgresMlEmbeddingClient配置选项，比如要使用的模型等；

  • 启动时，可将PostgresMlEmbeddingOptions传递给PostgresMlEmbeddingClient
    构造方法，以配置用于所有嵌入请求embedding request的默认选项；

  • 在运行时，可用EmbeddingRequest中的PostgresMlEmbeddingOptions覆盖默认选项；

var strList = List.of("Hello World",
	"World is big and salvation is near");

var embeddingResponse = embeddingClient.call(
	new EmbeddingRequest(strList,
		PostgresMlEmbeddingOptions.builder()
			.withTransformer("intfloat/e5-small")
            .withVectorType(VectorType.PG_ARRAY)
            .withKwargs(Map.of("device", "gpu"))
    	.build()));

4. Usage

spring:
  ai:
    postgresml:
      embedding:
        options:
          transformer: distilbert-base-uncased
          vectorType: PG_ARRAY
          metadataMode: EMBED
          kwargs:
            device: cpu
@RestController
public class EmbeddingController {

    private final EmbeddingClient embeddingClient;

    @Autowired
    public EmbeddingController(EmbeddingClient embeddingClient) {
        this.embeddingClient = embeddingClient;
    }

    @GetMapping("/ai/embedding")
    public Map embed(@RequestParam(value = "message",
    	defaultValue = "Tell me a joke") String message) {
        var embeddingResponse = this.embeddingClient
        	.embedForResponse(List.of(message));
        return Map.of("embedding", embeddingResponse);
    }
}
