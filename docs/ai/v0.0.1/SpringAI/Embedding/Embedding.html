1. Intro

  • EmbeddingClient旨在与AI和ML中的embedding model直接集成，
    主要功能：将文本转换为数字向量，通常称为嵌入embeddings，embeddings对如
    语义分析semantic analysis和文本分类text classification等各种任务至关重要；

  • EmbeddingClient设计主要目标：

    • A：可移植性Portability：接口确保跨各种嵌入模型的轻松适应性，
      允许工程师再不同的嵌入技术或模型间切换，只需少许代码更改；

    • B：简单性Simplicity：EmbeddingClient简化将文本转换为嵌入的过程，
      如embed(String text)和embed(Document document)，它消除处理原始文本数据
      和嵌入算法的复杂性，这使工程师甚至AI新手，无需深入研究底层机制；

2. Api Intro

  • ModelClient，EmbeddingClient，EmbeddingRequest，EmbeddingResponse；

  • Embedding Api构建在通用Spring AI模型Api之上，故EmbeddingClient接口扩展
    ModelClient接口，该接口提供一组与AI模型交互的标准方法，EmbeddingRequest和
    EmbeddingResponse类从ModelRequest和ModelResponse扩展，封装嵌入模型的输入和输出；

3. EmbeddingClient

  • embed()提供将文本转换为embeddings，参数接受单个字符串，结构化文档对象，文本列表；

  • 以上都围绕call()实现，call()是调用嵌入模型(embedding model)的主要方法；
    通常embedding返回double列表，以数字向量格式表示embeddings；

  • embedForResponse()提供更全面的输出，可能包括embeddings的附加(additional)信息，
    dimensions()是工程师快速确定嵌入向量大小的方便工具，对理解嵌入空间和后续处理步骤很重要；

public interface EmbeddingClient
	extends ModelClient<EmbeddingRequest,EmbeddingResponse> {

	@Override
	EmbeddingResponse call(EmbeddingRequest request);


	/**
	 * Embed given document's content into vector
	 * @param document document to embed
	 * @return embedded vector.
	 */
	List<Double> embed(Document document);

	/**
	 * Embed given text into vector
	 * @param text text to embed
	 * @return embedded vector
	 */
	default List<Double> embed(String text) {
		Assert.notNull(text, "Text must not be null");
		return this.embed(List.of(text)).iterator().next();
	}

	/**
	 * Embed batch of text into vector
	 * @param textList list of text to embed
	 * @return list of list of embedded vectors
	 */
	default List<List<Double>> embed(List<String> textList) {
		Assert.notNull(texts, "Texts must not be null");
		return this.call(new EmbeddingRequest(texts, EmbeddingOptions.EMPTY))
			.getResults()
			.stream()
			.map(Embedding::getOutput)
			.toList();
	}

	/**
	 * Embed batch of text into vectors and return {@link EmbeddingResponse}
	 * @param textList list of text to embed
	 * @return embedding response
	 */
	default EmbeddingResponse embedForResponse(List<String> textList) {
		Assert.notNull(texts, "Texts must not be null");
		return this.call(new EmbeddingRequest(texts, EmbeddingOptions.EMPTY));
	}

	/**
	 * @return number of dimension of embedded vector,
	 * It's generative specific.
	 */
	default int dimensions() {
		return embed("Test String").size();
	}

}

5. EmbeddingResponse

  • EmbeddingResponse类保存AI模型的输出，每个Embedding实例都包含来自单个文本输入的结果向量数据；

  • EmbeddingResponse类还携带有关AI模型响应的EmbeddingResponseMetadata元数据；

public class EmbeddingResponse implements ModelResponse<Embedding> {
	private List<Embedding> embeddingList;
	private EmbeddingResponseMetadata metadata
		= new EmbeddingResponseMetadata();
}
