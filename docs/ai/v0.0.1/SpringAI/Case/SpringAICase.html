1. Development Environment

  • Win 11 Pro，Jdk 22，Boot 3.2.X，Maven 3.9.0+；

  • Ram：16G(Instructor)，8G，16G+(Prefer)，OpenAI API_KEY；

  • https://github.com/spring-projects/spring-ai/blob/main/spring-ai-bom/pom.xml

3. Image Generation Vision

  • capability to generate image from text prompt

  • 目前Spring AI仅支持creating image from text

  • Available Image Model：dall-e-2，dall-e-3

3.1. Igv Step

  • A：create new image from text

  • B：edit image：enhance or remove/add object)

  • C：create image variation

3.2. dall-e-2

  • prompt up to 1000 character

  • size：256x256，512x512，1024x1024

3.3. dall-e-3

  • prompt up to 4000 character

  • size：1024x1024，1792x1024，1024x1792

3.4. Image Response Format

  • url：URL valid for 60 min after request

  • b64_json：Base 64 encode value in json field

4. TTS Text To Speech

Entry Memo

Input

text limit 4096 character

Model

tts-1(default，fast)，tts-1-hd(slower with better quality)

Voice Style

ally，echo，fable，onyx，nova，shimmer

Response Format

mp3(default)，opus，aac，flac，wav，pcm

Speed

range 0.25 - 4.0，default 1.0

5. STT Speech To Text

  • 能将音频转录为文本，
    ability to transcribe audio to text

  • 转录的文本将与音频的语言匹配，
    transcribe text match audio language

  • 可将不同语言的音频翻译成英语，
    translate different language audio into English

  • 提示可提高质量，并为不常用单词提供正确的拼写，
    Prompting improve quality and provide correct spelling

  • GPT-4可与提示合用，以对生成的文本执行后处理，
    GPT-4 use with prompt to perform post processing on generated text

Entry Memo

Support Input File

flac，mp3，mp4，mpeg，mpga，m4a，ogg，wav，webm

Available Model

whisper-1，whisper 2(open source)

Response Format

json，text，srt，verbose_json，vtt

  • Use System Message to Improve Model Response
