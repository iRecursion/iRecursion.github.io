1. Introduction

  • Milvus是向量数据库，在数据科学和ML引起极大的关注，突出特点如对向量索引和查询的支持，
    Milvus采用最先进的(state-of-the-art)尖端(cutting-edge)算法加速搜索，
    即使处理大数据集，也能非常高效地检索相似的向量；

2. Milvus Setup

  • https://milvus.io/docs

  • https://github.com/milvus-io/milvus/releases

  • https://github.com/zilliztech/attu/releases

  • https://github.com/zilliztech/attu

  • https://github.com/zilliztech/VectorDBBench

  • wget https://github.com/milvus-io/milvus/releases/download/v2.3.12/milvus-standalone-docker-compose.yml

  • mv milvus-standalone-docker-compose.yml docker-compose.yaml

  • 删除网络，docker-compose up -d

  • docker-compose down && rm -rf volumes

  • http://192.168.0.123:9001：minioadmin - minioadmin

  • https://milvus.io/docs/install_standalone-docker.md

  • 19530端口：gRpc和Restful Api，连接到Milvus服务器的默认端口；

  • 9091端口：是管理端口，用于metric collection，
    pprof profiling和health probe within K8S；

  • Milvus默认凭证：root - Milvus，最大连接数65536

  • 备份工具：https://github.com/zilliztech/milvus-backup

  • Troubleshooting：http://192.168.0.123:19530

docker system prune --all --force --volumes

3. Gui Attu

  • http://192.168.0.123:8000

  • MILVUS_URL可选，忌用127.0.0.1或localhost

docker run \
	--name attu \
	-p 8000:3000 \
	--privileged=true \
	--net network-common \
	-e MILVUS_URL=192.168.0.123:19530 \
	-itd zilliz/attu:v2.3.8

5. Yaml

spring:
  ai:
    vectorstore:
      milvus:
        database-name: default
        collection-name: vector_store
        embedding-dimension: 1536
        index-type: IVF_FLAT
        metric-type: cosine
        client:
          host: 192.168.0.123
          port: 19530
          username: root
          password: Milvus

6. Usage

@Autowired
VectorStore vectorStore;

List <Document> documentList = List.of(
    new Document("Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!!", Map.of("meta1", "meta1")),
    new Document("The World is Big and Salvation Lurks Around the Corner"),
    new Document("You walk forward facing the past and you turn back toward the future.", Map.of("meta2", "meta2")));

vectorStore.add(documentList);
List<Document> resultList = vectorStore
	.similaritySearch(SearchRequest.query("Spring").withTopK(5));

7. Metadata Filtering

  • text expression language方式：

vectorStore.similaritySearch(
	SearchRequest.defaults()
	    .withQuery("The World")
	    .withTopK(TOP_K)
	    .withSimilarityThreshold(SIMILARITY_THRESHOLD)
	    .withFilterExpression("author in ['john', 'jill'] && article_type == 'blog'"));

  • programmatically Filter.Expression Dsl方式：

var builder = new FilterExpressionBuilder();

vectorStore.similaritySearch(SearchRequest.defaults()
    .withQuery("The World")
    .withTopK(TOP_K)
    .withSimilarityThreshold(SIMILARITY_THRESHOLD)
    .withFilterExpression(b.and(
        builder.in("john", "jill"),
        builder.eq("article_type", "blog")).build()));
