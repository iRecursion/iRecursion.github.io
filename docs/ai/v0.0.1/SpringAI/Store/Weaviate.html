1. Introduction

  • Weaviate是开源向量数据库，允许存储ML模型的数据对象和向量嵌入，
    并无缝扩展到数十亿(scale seamlessly into billion)个数据对象中，
    提供存储文档嵌入、内容和元数据及搜索嵌入(包括元数据过滤)的工具；

2. Prerequisite

  • EmbeddingClient实例计算文档嵌入document embeddings，可用选项：

    • PostgresML Embedding Client；

    • Transformer Embedding：计算本地环境中的嵌入，
      按Onnx Transformer Embedding instruction操作；

5. Usage

@Bean
VectorStore vectorStore(EmbeddingClient embeddingClient) {
	var config = WeaviateVectorStoreConfig.builder()
		.withScheme("http")
		.withHost("192.168.0.123:8080")

		//define metadata field to used in similarity search filter
		.withFilterableMetadataFields(List.of(
	        MetadataField.text("country"),
	        MetadataField.number("year"),
	        MetadataField.bool("active")))

		//consistency level can：ONE,QUORUM,ALL
		.withConsistencyLevel(ConsistentLevel.ONE)
		.build();
	return new WeaviateVectorStore(config,embeddingClient);
}

  • 必须明确explicitly列出filter expression中使用的任何元数据键
    metadata key的所有元数据字段名称和类型(BOOLEAN、TEXT或NUMBER)，
    text("country"),number("year"),bool("active")；

  • 若filterable metadata field expand with new entry，
    那必须(re)upload/update documents with this metadata；

  • 可在没明确定义的情况下使用以下Weaviate系统元数据字段：
    id、_creationTimeUnix和_lastUpdateTimeUnix；

List<Document> documentList = List.of(
   new Document("Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!!", Map.of("country", "UK", "active", true, "year", 2020)),
   new Document("The World is Big and Salvation Lurks Around the Corner", Map.of()),
   new Document("You walk forward facing the past and you turn back toward the future.", Map.of("country", "NL", "active", false, "year", 2023)));

vectorStore.add(List.of(document));

List<Document> resultList = vectorStore.similaritySearch(

6. Metadata Filtering

  • text expression language方式：

vectorStore.similaritySearch(SearchRequest
	.query("The World")
  	.withTopK(TOP_K)
  	.withSimilarityThreshold(SIMILARITY_THRESHOLD)
  	.withFilterExpression("country in ['UK', 'NL'] && year >= 2020"));

  • programmatically expression Dsl方式：

var builder = new FilterExpressionBuilder();

vectorStore.similaritySearch(SearchRequest
  	.query("The World")
	.withTopK(TOP_K)
  	.withSimilarityThreshold(SIMILARITY_THRESHOLD)
  	.withFilterExpression(b.and(
     	builder.in("country", "UK", "NL"),
     	builder.gte("year", 2020)).build()));

  • 可移植的过滤器表达式portable filter expression会自动
    转换为专有的(proprietary)Weaviate where过滤器，如：
    country in ['UK', 'NL'] && year >= 2020

  • 转换为：Weaviate GraphQL where filter expression：

operator:And
   operands:
      [{
         operator:Or
         operands:
            [{
               path:["meta_country"]
               operator:Equal
               valueText:"UK"
            },
            {
               path:["meta_country"]
               operator:Equal
               valueText:"NL"
            }]
      },
      {
         path:["meta_year"]
         operator:GreaterThanEqual
         valueNumber:2020
      }]
