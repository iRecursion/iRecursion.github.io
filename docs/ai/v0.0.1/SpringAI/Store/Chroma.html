1. Introduction

  • 设置Chroma VectorStore以存储document embedding并执行similarity search；

  • Chroma是embedding数据库，以存储document embedding，content
    和metadata，可通过embedding，包括metadata filtering来搜索；

  • https://www.trychroma.com

  • 启动时，ChromaVectorStore会创建所需的集合(若未设置)

2. Start Chroma

docker run \
	--name chroma \
	-p 8000:8000 \
	--privileged=true \
	--net network-common \
	-itd chromadb/chroma:0.4.24

  • http://192.168.0.123:8000/api/v1

3. Dependency

<dependency>
  	<groupId>org.springframework.ai</groupId>
  	<artifactId>spring-ai-chroma-store</artifactId>
</dependency>

  • 使用正确的(proper)ChromaDB授权配置创建
    RestTemplate实例，并使用它来创建ChromaApi实例：

@Bean
RestTemplate restTemplate() {
   return new RestTemplate();
}

@Bean
ChromaApi chromaApi(RestTemplate restTemplate) {
   String chromaUrl = "http://192.168.0.123:8000";
   ChromaApi chromaApi = new ChromaApi(chromaUrl, restTemplate);
   return chromaApi;
}

  • 对使用静态Api Token Authentication保护的ChromaDB，请使用
    ChromaApi.withKeyToken(<Your Token Credential>)来设置凭据；

  • 对使用Basic Authentication保护的ChromaDB，请使用
    ChromaApi.withBasicAuth(<userName>,<password>)来设置凭据；

4. Usage

@Bean
VectorStore chromaVectorStore(EmbeddingClient client,ChromaApi api) {
	return new ChromaVectorStore(client,api,"TestCollection");
}
List<Document> documentList = List.of(
	new Document("Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!!", Map.of("meta1", "meta1")),
	new Document("The World is Big and Salvation Lurks Around the Corner"),
	new Document("You walk forward facing the past and you turn back toward the future.", Map.of("meta2", "meta2")));

vectorStore.add(documentList);

List<Document> resultList = vectorStore.similaritySearch("Spring");

5. Metadata Filtering

  • text expression language方式：

vectorStore.similaritySearch(
	SearchRequest.defaults()
	    .withQuery("The World")
	    .withTopK(TOP_K)
	    .withSimilarityThreshold(SIMILARITY_THRESHOLD)
	    .withFilterExpression("author in ['john', 'jill'] && article_type == 'blog'"));

  • 编程式：Filter.Expression Dsl：

var builder = new FilterExpressionBuilder();

vectorStore.similaritySearch(SearchRequest.defaults()
	.withQuery("The World")
	.withTopK(TOP_K)
	.withSimilarityThreshold(SIMILARITY_THRESHOLD)
	.withFilterExpression(b.and(
        builder.in("john", "jill"),
        builder.eq("article_type", "blog")).build()));

  • 这些便携式portable过滤器表达式自动转换为专有的Chroma where过滤器表达式，
    如portable filter expression：

  • author in ['john', 'jill'] && article_type == 'blog'

  • 转换为：proprietary Chroma format：

{"$and":[
	{"author": {"$in": ["john", "jill"]}},
	{"article_type":{"$eq":"blog"}}]
}
