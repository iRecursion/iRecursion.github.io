1. Introduction

  • 引导设置RedisVectorStore，存储文档嵌入并执行相似性搜索；
    store document embeddings and perform similarity search；

  • Redis Search and Query扩展Redis OSS的核心功能，允许将Redis用作向量数据库：

    • 将向量和相关元数据存储在hash或Json文档中

    • 检索向量 Retrieve Vector

    • 执行向量搜索 Perform Vector Search

2. Prerequisite

  • EmbeddingClient实例来计算文档嵌入(document embeddings)：

  • Transformers Embedding：计算本地环境中的嵌入，
    按Onnx Transformer Embedding instruction(说明)进行操作；

  • docker镜像redis/redis-stack:tagVersion

3. Dependency

  • Embedding Client boot starter，
    required for calculating embeddings.

  • Transformers Embedding (Local) and follow
    Onnx Transformer Embedding instruction；

  • Redis Vector Store and Jedis Dependency

4. Usage

@Configuration
public class RedisVectorConfig {

	@Bean
	VectorStore vectorStore(EmbeddingClient embeddingClient) {
		var config = RedisVectorStoreConfig.builder()
			.withURI("redis://localhost:6379")
			//define metadata field to used in similarity search filter
			.withMetadataFields(
				MetadataField.tag("country"),
				MetadataField.numeric("year"))
			.build();
		return new RedisVectorStore(config,embeddingClient);
	}

}

  • 将RedisVectorStore创建为Bean更方便，也更可取，但若手动创建，那在设置属性后
    和使用客户端前，必须调用RedisVectorStore.afterPropertiesSet()；

  • 必须明确列出过滤表达式中使用的任何元数据字段的所有元数据字段名称和类型(TAG、TEXT或NUMERIC)
    withMetadataFields注册的过滤元数据字段：TAG类型的国家/地区，NUMERIC类型的年份；

  • 然后在主代码中，创建文档：

List<Document> documentList = List.of(
   new Document("Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!!", Map.of("country", "UK", "year", 2020)),
   new Document("The World is Big and Salvation Lurks Around the Corner", Map.of()),
   new Document("You walk forward facing the past and you turn back toward the future.", Map.of("country", "NL", "year", 2023)));

  • 最后这样查询：将返回：Spring AI rocks!!

List<Document> resultList = vectorStore.similaritySearch(
	SearchRequest.query("Spring").withTopK(5));

5. Metadata Filtering

  • 可使用RedisVectorStore来利用通用的，可移植的元数据过滤器，
    leverage generic，portable metadata filter；

  • 可使用文本表达式语言：

vectorStore.similaritySearch(
	SearchRequest.query("The World")
		.withTopK(TOP_K)
		.withSimilarityThreshold(SIMILARITY_THRESHOLD)
		.withFilterExpression("country in ['UK', 'NL'] && year >= 2020"));

  • 或以编程方式使用表达式DSL：

vectorStore.similaritySearch(
	SearchRequest.query("The World")
		.withTopK(TOP_K)
		.withSimilarityThreshold(SIMILARITY_THRESHOLD)
		.withFilterExpression(
			b.and(b.in("country", "UK", "NL"),
		          b.gte("year", 2020)).build())
		);

  • 可移植的过滤器表达式portable filter expression会自动转换为Redis搜索查询，
    如：country in ['UK', 'NL'] && year >= 2020

  • 已转换为Redis查询：@country:{UK | NL} @year:[2020 inf]
    https://redis.io/docs/interact/search-and-query/query/
