1. Prerequisite

  • 设置Qdrant VectorStore，以存储document embeddings
    和执行similarity search，Qdrant是开源，高性能的向量搜索引擎/数据库；

http api，monitoring health and metric endpoint

6334

gRPC api

6335

distribute deployment

4. Usage

@Autowired
VectorStore vectorStore;

List <Document> documentList = List.of(
    new Document("Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!! Spring AI rocks!!", Map.of("meta1", "meta1")),
    new Document("The World is Big and Salvation Lurks Around the Corner"),
    new Document("You walk forward facing the past and you turn back toward the future.", Map.of("meta2", "meta2")));

//add document list to Qdrant
vectorStore.add(List.of(documentList));

//retrieve document similar to query
List<Document> resultList = vectorStore
	.similaritySearch(SearchRequest.query("Spring").withTopK(5));

5. Metadata Filtering

  • 可使用Qdrant向量存储利用通用的，可移植的元数据过滤器；
    leverage generic，portable metadata filter；

  • text expression language 方式：

var expression = "author in ['john', 'jill'] && article_type == 'blog'";

vectorStore.similaritySearch(
    SearchRequest.defaults()
    	.withQuery("The World")
	    .withTopK(TOP_K)
	    .withSimilarityThreshold(SIMILARITY_THRESHOLD)
	    .withFilterExpression(expression));

  • programmatically using Filter.Expression Dsl：

var builder = new FilterExpressionBuilder();

vectorStore.similaritySearch(SearchRequest.defaults()
    .withQuery("The World")
    .withTopK(TOP_K)
    .withSimilarityThreshold(SIMILARITY_THRESHOLD)
    .withFilterExpression(b.and(
        builder.in("john", "jill"),
        builder.eq("article_type", "blog")).build()));

  • 这些过滤器表达式都可转换为等效的Qdrant过滤器；

7. Qdrant Entry

7.1. Initialize Client

import io.qdrant.client.QdrantClient;
import io.qdrant.client.QdrantGrpcClient;

var client = new QdrantClient(
    QdrantGrpcClient.newBuilder("192.168.0.123",6334,false).build());

7.2. Create Collection

import io.qdrant.client.grpc.Collections.Distance;
import io.qdrant.client.grpc.Collections.VectorParams;

client.createCollectionAsync("test_collection",
    VectorParams.newBuilder().setDistance(Distance.Dot)
    	.setSize(4).build()).get();

7.3. Add Vector

import java.util.List;
import java.util.Map;
import java.util.concurrent.ExecutionException;

import io.qdrant.client.PointIdFactory;
import io.qdrant.client.QdrantClient;
import io.qdrant.client.QdrantGrpcClient;
import io.qdrant.client.ValueFactory;
import io.qdrant.client.VectorsFactory;
import io.qdrant.client.grpc.Points;
import io.qdrant.client.grpc.Points.PointStruct;
import io.qdrant.client.grpc.Points.UpdateResult;

var gRpcClient = QdrantGrpcClient
	.newBuilder("192.168.0.123",6334,false).build();
var client = new QdrantClient(gRpcClient);

List<Points.PointStruct> psList = List.of(
	PointStruct.newBuilder()
		.setId(PointIdFactory.id(1))
		.setVectors(VectorsFactory.vectors(0.05f,0.61f,0.76f,0.74f))
		.putAllPayload(Map.of("city", ValueFactory.value("Berlin")))
		.build(),

	PointStruct.newBuilder()
		.setId(PointIdFactory.id(2))
		.setVectors(VectorsFactory.vectors(0.19f,0.81f,0.75f,0.11f))
		.putAllPayload(Map.of("city", ValueFactory.value("London")))
		.build(),

	PointStruct.newBuilder()
		.setId(PointIdFactory.id(4))
		.setVectors(VectorsFactory.vectors(0.36f,0.55f,0.47f,0.94f))
		.putAllPayload(Map.of("city", ValueFactory.value("Moscow")))
		.build());

UpdateResult operationInfo = client
	.upsertAsync("test_collection",psList).get();

System.out.println(operationInfo);

7.4. Run Query

var gRpcClient = QdrantGrpcClient.newBuilder("192.168.0.123",6334,false).build();
var client = new QdrantClient(gRpcClient);

List<ScoredPoint> searchResult = client.searchAsync(
    SearchPoints.newBuilder()
        .setCollectionName("test_collection")
        .setLimit(3)
        .addAllVector(List.of(0.2f,0.1f,0.9f,0.7f))
        .setWithPayload(WithPayloadSelectorFactory.enable(true))
        .build()).get();

System.out.println(searchResult);

7.5. Add Filter

var gRpcClient = QdrantGrpcClient.newBuilder("192.168.0.123",6334,false).build();
var client = new QdrantClient(gRpcClient);

List<ScoredPoint> searchResult = client.searchAsync(
    SearchPoints.newBuilder()
        .setCollectionName("test_collection")
        .setLimit(3)
        .setFilter(Filter.newBuilder().addMust(
        	ConditionFactory.matchKeyword("city", "London")))
        .addAllVector(List.of(0.2f, 0.1f, 0.9f, 0.7f))
        .setWithPayload(WithPayloadSelectorFactory.enable(true)).build()).get();

System.out.println(searchResult);
