1. Intro

  • SpringAI支持ZhiPu图像生成模型CogView；

export SPRING_AI_ZHIPU_AI_API_KEY=<INSERT KEY HERE>
<dependency>
    <groupId>org.springframework.ai</groupId>
    <artifactId>spring-ai-zhipuai-spring-boot-starter</artifactId>
</dependency>

2. Image Generation Property

Property Description Default Value

前缀：spring.ai.zhipuai.image.

enabled

是否启用ZhiPu图像模型

true

base-url

可选，覆盖spring.ai.zhipuai.base-url

-

api-key

可选，覆盖spring.ai.zhipuai.api-key

-

options.model

用于图像生成的模型

cogview-3

options.user

end-user的唯一标识符，监控和检测滥用行为，
unique identifier，monitor and detect abuse

-

 
spring:
  ai:
    zhipuai:
      image:
        enabled: true
        base-url:
        api-key:
        options:
          model: cogview-3
          user:

4. Retry Property

Property Description Default Value

前缀：spring.ai.retry.

max-attempts

retry attempt maximum number

10

backoff.initial-interval

initial sleep duration for exponential backoff policy

2(second)

backoff.multiplier

backoff interval multiplier

5

backoff.max-interval

maximum backoff duration

3(minute)

on-client-errors

false：throw NonTransientAiException，
don’t attempt retry for 4xx client error code

false

exclude-on-http-codes

not trigger retry http status code list，
e.g. to throw NonTransientAiException

empty

on-http-codes

trigger retry http status code list，
e.g. to throw TransientAiException

empty

5. Runtime Option

  • ZhiPuAiImageOptions，ImagePrompt，
    ZhiPuAiImageModel(ZhiPuAiImageApi api)；

  • withDefaultOptions(ZhiPuAiImageOptions option)；
    alternatively：spring.ai.zhipuai.image.options.*；

var response = zhiPuAiImageModel.call(
	new ImagePrompt("A light cream colored mini golden doodle",
	ZhiPuAiImageOptions.builder()
		.withQuality("hd")
		.withN(4)
		.withHeight(1024)
		.withWidth(1024).build())

);
