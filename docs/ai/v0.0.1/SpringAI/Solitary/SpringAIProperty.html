4.4. Postgres

Property Description Default Value

前缀：spring.ai.vectorstore.pgvector.

index-type

  • 最近邻居(NN)搜索索引类型;

  • NONE:精确exact最近邻居搜索;

  • IVFFlat:将向量拆分为list,
    再搜最接近查询向量的列表子集;

  • HNSW:创建多层图形;

  • IVFFlat比HNSW构建更快,占内存更少,查询性能较低,
    即speed-recall tradeoff,HNSW则相反;

  • 另HNSW没IVFFlat那样的训练步骤,
    故可在表中没任何数据情况下创建索引;

HNSW

dimensions

  • Embedding dimension;

  • 若没指定，PgVectorStore将从提供的
    EmbeddingClient中检索dimension;

  • 建表时，将Dimension设置为embedding column;

  • 若改dimension，则还须重新创建vector_store表;

  • -1表示无效的embedding dimension;

-1

distance-type

  • 搜索距离类型;

  • 若向量normalize to长度1，
    则可用EUCLIDEAN_DISTANCE或
    NEGATIVE_INNER_PRODUCT,以获最佳性能

COSINE_DISTANCE

remove-existing-vector-store-table

启动时删除现有vector_store表

false

 
spring:
  ai:
    vectorstore:
      pgvector:
        index-type: HNSW
        dimensions: -1
        distance-type: COSINE_DISTANCE
        remove-existing-vector-store-table: false

4.5. Weaviate

Property Description Default Value

http

Weaviate服务器主机

scheme

api-key

Weaviate服务器认证的Api Key

objectClass

SpringAiWeaviate

consistency-level

参考类ConsistentLevel:one,quorum,all

one

headers

头部信息，Map

filter-field

过滤字段名和类型(text,number,boolean)

  • https://weaviate.io/developers/weaviate/concepts/
    replication-architecture/consistency#tunable-consistency-strategies

spring:
  ai:
    vectorstore:
      weaviate:
        scheme: http
        host: 192.168.0.123:8080
        api-key:
        object-class: SpringAiWeaviate
        #one,quorum,all
        consistency-level: one
        headers:
          headerMapKey: headerMapValue
        filter-field:
          #text,number,boolean
          fieldName: text
          fieldAge: number
          fieldSex: boolean
