1. Introduction

  • OutputParser接口允许结构化输出，如将输出映射到Java类或来自AI模型基于字符串的输出的值数组；
    array of value from String based ouput of AI Model

  • 可将其视为类似于Spring JDBC的RowMapper或ResultSetExtractor，
    开发人员希望快速将AI模型的结果转换为可传递给应用程序中其它函数和方法的数据类型；

  • turn result from AI model into data type that
    pass to other function and method in application；

public interface OutputParser<T> extends Parser<T>,FormatProvider {}

@FunctionalInterface
public interface Parser<T> {

	T parse(String text);

}

public interface FormatProvider {

	String getFormat();

}

  • Parser接口解析文本字符串以生成T类型的实例；
    Parser parse text str to produce instance of type T；

  • FormatProvider为AI模型提供文本指令，以格式化输出，
    从而使其可由解析器解析为类型T，文本指令通常追加在AI模型的用户输入的末尾；

2. Available RI

2.1. BeanOutputParser

  • 为Java类指定Json Schema，并使用Json Schema规范的DRAFT_2020_12，
    AI模型的Json输出被反序列化为Java对象，即JavaBean

2.2. MapOutputParser

  • 类似于(similar to)BeanOutputParser，
    但Json负载(payload)被反序列化为Map＜String,Object＞实例

2.3. ListOutputParser

  • 将输出指定为逗号(comma)分隔的列表；

3. Example Usage

class ActorFilm {
	private String actor;
	private List<String> movieList;

	@Override
	public String toString() {
		return "ActorFilm [actor=" + actor + ", movieList=" + movieList + "]";
	}

	public String getActor() {
		return actor;
	}
	public List<String> getMovieList() {
		return movieList;
	}

}
import java.util.List;
import java.util.Map;

import org.springframework.ai.chat.ChatClient;
import org.springframework.ai.chat.Generation;
import org.springframework.ai.parser.BeanOutputParser;
import org.springframework.ai.prompt.Prompt;
import org.springframework.ai.prompt.PromptTemplate;
import org.springframework.beans.factory.annotation.Autowired;
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.RequestParam;

public class OutputParserEntry {

	@Autowired
	private ChatClient aiClient;

	@GetMapping("/ai/output")
	public ActorFilm generate(@RequestParam(value = "actor",
		defaultValue = "Elf") String actor) {
	    var outputParser = new BeanOutputParser<>(ActorFilm.class);

		String userMessage = """
		    generate filmography for actor {actor}.
		    {format}
		    """;

	    PromptTemplate promptTemplate = new PromptTemplate(userMessage,
	   		Map.of("actor",actor,"format",outputParser.getFormat()));

	    Prompt prompt = promptTemplate.create();
	    Generation generation = aiClient.generate(prompt).getGeneration();

	    ActorFilm actorFilm = outputParser.parse(generation.getContent());
	    return actorFilm;
	}

}
