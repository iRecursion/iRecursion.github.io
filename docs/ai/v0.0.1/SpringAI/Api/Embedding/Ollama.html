1. Intro

  • https://ollama.ai

  • LLM：Large Language Model，大型语言模型；

  • Ollama可在本地运行各种大型语言模型(LLM)，并从中生成嵌入embeddings，
    Spring AI通过OllamaEmbeddingClient支持Ollama文本嵌入；

  • embedding是浮点数字的向量(列表)，vector (list) of floating point number，
    两个向量间的距离测量measure它们的相关性relatedness，距离小即高相关性，距离大即低相关性；

  • 参考OllamaChat.adoc，先搭建起Ollama本地运行环境；

<dependency>
	<groupId>org.springframework.ai</groupId>
   	<artifactId>spring-ai-ollama-spring-boot-starter</artifactId>
</dependency>

2. Embedding Properties

  • Embedding Properties

  • spring.ai.ollama.embedding.options.*属性用于配置所有嵌入请求(embedding request)
    的默认选项，被用作OllamaEmbeddingClient#withDefaultOptions的实例；

  • spring.ai.ollama.embedding.options.*属性基于
    Ollama Valid Parameter and Value and Ollama Type；

  • 所有以spring.ai.ollama.embedding.options为前缀的属性都可在运行时
    通过向EmbeddingRequest调用添加特定于请求的embedding options来覆盖；

3. Embedding Option

OllamaOptions.java

  • OllamaOptions.java提供Ollama配置，如使用的模型(model to use)，
    低级别GPU和CPU调优(low level GPU and CPU tunning)等；

  • 默认选项也可使用spring.ai.ollama.embedding.options属性配置，

  • 在启动时，用OllamaEmbeddingClient#withDefaultOptions()
    来配置用于所有嵌入请求embedding request的默认选项；

  • 在运行时，可用OllamaOptions实例作为EmbeddingRequest的一部分来覆盖默认选项，如：

var strList = List.of("Hello World","World is big and salvation is near");
var embeddingResponse = embeddingClient.call(
    new EmbeddingRequest(strList,OllamaOptions.create()
    	.withModel("Different-Embedding-Model-Deployment-Name"));

4. Usage

@RestController
public class EmbeddingController {

    private final EmbeddingClient embeddingClient;

    @Autowired
    public EmbeddingController(EmbeddingClient embeddingClient) {
        this.embeddingClient = embeddingClient;
    }

    @GetMapping("/ai/embedding")
    public Map embed(@RequestParam(value = "message",
    	defaultValue = "Tell me a joke") String message) {
        var embeddingResponse = this.embeddingClient
        	.embedForResponse(List.of(message));
        return Map.of("embedding",embeddingResponse);
    }
}
