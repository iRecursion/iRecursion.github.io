2. Grammar

  • 巴克斯-诺尔形式中(Backus-Naur form)；

  • ::= 表示一类符号(symbol category)，
    可能是右边列出的可能的符号序列(possible symbol sequence)之一；

v

::=

[A-Za-z]，greek letters，number，other constant symbol

u

::=

sqrt，text，bb，other unary symbol for font command

b

::=

frac，root，stackrel，other binary symbol

l

::=

(，[，{，(:，{:，other left bracket

r

::=

)，]，}，:)，:}，other right bracket

S

::=

v，lEr，uS，bSS：Simple Expression

i

::=

S_S，S^S，S_S^S，S：Intermediate Expression

E

::=

IE，I/I：Expression
