2. Rational Number

  • 相反数：数值相同，符号相反的两个数，如往左走3米，往右走3米；

  • 有理数：整数和分数(有限小数和无限小数)，有理数都能转换成整数之比，又称可比数；

  • 有理数 = 整数 + 分数 = 正整数 + 负整数 + 正分数 + 负分数 + 0；

  • 无限不循环小数，不是有理数，如π；

  • 数轴三要素：原点，正方向，单位长度；

  • 绝对值：某数到原点的距离，绝对值的非负性；

  • 算加减乘除中，先将减法换成加法，除法换成乘法，再随意计算，否则只能同类从左到右；

  • An：a为底数，n为指数，An就是乘方，结果叫幂；

  • 有理数运算法则：先乘方，其次乘除，最后加减，另需考虑括号；

  • 科学记数法：整数部分：大于0的一位数，如1.5 * 103，也可四舍五入取近似数(值)；

  • 绝对值化简：当绝对值内小于零时，需取反，如a > b，- |a - b| = - a + b，
    a + c < 0，则|a + c| = -a - c；

3. Integral Expression

3.1. Monomial

  • 整式：单项式和多项式的统称，有理式的一部分，
    可包含加减乘除乘方运算，但在整式中除数不能含有字母；

  • 单项式：数与字母的乘积，结果数字在前，字母在后，如3A；

  • 而3x + 4y不是单项式，而x/2是单项式，2/x则不是单项式，是除法运算；

  • 单项式包括单独的数字或字母，如2，x；

  • 单项式的系数和次数，系数是单项式数的部分，次数是单项式中字母的个数，
    如3ab：系数为3，次数为2；没字母的次数为0；

3.2. Polynomial

  • 多项式：几个单项式的和，如3x + 5y，X2 + 2X + 18

  • 多项式的项：2X3 -3X2 + X -1

  • 三次项：2X3，次数为3，系数为2，二次项：-3X2，次数为2，系数为-3，
    一次项：X，次数为1，系数为1，常数项为-1，次数无，系数为-1；

  • 多项式书写规范：按X或按Y的次数降幂或升幂排列；

3.3. Similar Item

  • 若两单项式，他们所含的字母相同，且相同字母的指数也分别相同，那就称这两个单项式为同类项；

  • 如4y与5y，100ab与14ab，6c与6c，另所有常数项都是同类项，常数项也称数字因数；

  • 合并同类项：字母不变，系数加减，另需考虑括号和乘法分配律；
