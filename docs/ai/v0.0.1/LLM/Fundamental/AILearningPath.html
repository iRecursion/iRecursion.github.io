1. 引言

  • 人工智能是数据科学的一部分，AI分狭义或弱人工智能(Narrow/Weak AI)，
    和通用或强人工智能(General/Strong AI)；

  • 狭义AI旨在执行特定任务，如语音识别；

  • 通用AI可执行人类可完成的任何智力(intellectual)任务

  • AI旨在构建智能机器，ML是AI子集，机器无需明确(explicitly)编程即可从数据中学习，
    DL是ML子集，其中神经网络模仿人脑来寻找模式(pattern)并做出决策；

  • 无监督学习和强化学些是AI和ML的两个关键领域，旨在模仿人类智能的各方面，

  • 无监督学习涉及在未标记的数据上训练算法，以独立识别模式(pattern)和结构(structure)，
    反映人类通常在没任何明确指导(explicit guidance)下通过观察周围的世界来学习；

  • 强化学习从行为心理学中获得灵感，因它围绕训练代理(agent)通过反复实验(trial and error)
    做出顺序(sequential)决策，像人类根据反馈调整和改进他们的行为一样；

  • 对有数学和编程基础的人，需6~12月持续学习来理解AI基础知识并熟悉ML过程，
    神经网络或强化学习等不同AI分值则需更深入，更专业的知识需再增加6~12月；
    学习AI需快速学习，持续学习，彻底(thoroughly)学习；

2. A 掌握必备技能

  • 基础技能：统计学和数学；核心技能：编程；基本概念：数据结构和操作(manipulation)

2.1. Statistic

  • 统计是ML支柱，理解统计度量(statistical measure)，分布(distribution)
    和偏差(deviation)或方差(variance)等概念，指导AI系统从数据进行推理；

2.2. Mathematic

  • 彻底掌握AI中ML算法的功能和执行，线性代数(linear algebra)、微积分(calculus)、
    概率论(probability theory)和优化方法(optimization)等数学基础，有效导航数组，
    执行复杂计算，理解数据中随机性和不规则性及最终优化ML模型以实现更改好的AI功能的关键任务；

2.3. Programming

  • 通常首选Python，R主要用于数据科学，
    Java和Cpp等提供高效的低级机器控制，同时推荐系统级Rust；

2.4. Data Structure

  • 数据结构也是AI基石，需以数组，栈，树等方式组织数据，将此与操作这些数据以执行
    所需操作的能力相结合，形成复杂的ML算法，并随后帮助AI模型理解数据的基础；

  • Array，LinkedList，Stack，Queue，BinaryTree，
    BinarySearchTree，Heap，Hashing，Graph，Matrix

3. B 培养专业的AI技能

3.1. Data Science

  • 熟练的数据科学家工具包包括数据预处理专业知识(expertise)；

  • 统计测试的熟练程度(adeptness in statistical testing)；

  • 熟练的数据可视化技能(adept data visualization skill)；

3.2. ML和DL in AI

  • ML和DL是AI的基石，ML让系统具有从经验中学习和改进的能力，
    是AI系统做出明智决策和预测的基石；

  • DL是ML子集，利用神经网络来模拟人类认知的复杂网络，
    使AI模型能自动发现数据中可能逃避传统编程范式中的复杂模式和特征；

4. C 熟悉基本的AI工具和包

4.1. Pandas

  • 数据操作和分析，提供强大的数据结构，有效存储和操作数据；

4.2. NumPy

  • 处理数组，线性代数，傅里叶变换Fourier Transform和矩阵Matrice

4.3. Scikit Learn

  • 支持向量机，随机森林，K近邻，还支持Python数字，科学库，如NumPy和SciPy

4.4. PyTorch

  • 构建ML和DL模型提供灵活性和速度，主要用于NLP；

4.5. Keras

  • 神经网络库，适合新手，易使用和理解

5. D AI学习规划

  • 1~3月学习数学，编程，数据结构和操作(manipulation)；

  • 4~6月沉浸学习AI和ML，如Scikit Learn和Keras；

  • 深入高级AI主题的专业化：PyTorch，图像识别和NLP；

  • 选择AI重点专业化(Specialization)与泛化(Generalization)：
    如ML，机器人Robotic，NLP，计算机视觉Computer Vision等；

  • 持续学习和探索，参与全球AI竞赛，跟上arxiv或Kaggle最新研究；
