1. 概述

  • 国家AI产业综合标准化体系建设指南

  • National AI Industry Comprehensive
    Standardization System Construction Guideline

2. 产业发展现状

  • AI产业链 = 基础层 + 框架层 + 模型层 + 应用层；

  • A：基础层：算力 + 算法 + 数据；

  • B：框架层：用于模型开发的深度学习框架和工具；

  • C：模型层：大模型等技术和产品；

  • D：应用层：AI技术在行业场景的应用

3. 总体要求

  • A：坚持创新驱动，B：坚持应用牵引，C：坚持产业协同，D：坚持开放合作；

4. 建设思路

4.1. AI 标准体系结构

  • AI标准体系结构 = 基础共性 + 基础支撑 + 关键技术
    智能产品与服务 + 赋能新型工业化 + 行业应用 + 安全/治理；

  • A：基础共性标准：AI的基础性、框架性、总体性标准；

  • B：基础支撑标准：规范数据、算力、算法等技术要求，为AI产业发展夯实技术底座；

  • C：关键技术标准：规范AI文本、语音、图像，及人机混合增强智能、
    智能体、跨媒体智能、具身智能等的技术要求，推动AI技术创新和应用；

  • D：智能产品与服务标准：规范由AI技术形成的智能产品和服务模式；

  • E：赋能新型工业化标准：规范AI技术赋能制造业全流程智能化以及重点行业智能升级的技术要求；

  • F：行业应用标准：规范AI赋能各行业的技术要求，
    为AI赋能行业应用，推动产业智能化发展提供技术保障；

  • G：安全/治理标准：规范AI安全、治理等要求，为AI产业发展提供安全保障；

4.2. AI 标准体系框架

  • AI标准体系框架 = 基础共性 + 基础支撑 + 关键技术
    智能产品与服务 + 赋能新型工业化 + 行业应用 + 安全/治理；
