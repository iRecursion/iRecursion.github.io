1. Intro

  • Principal/Senior AI Engineer

2. Key Responsibility

2.1. Design and Optimization

  • AI Model Design and Optimization：模型设计与优化

  • 领导AI算法的设计、实施和优化，使其在边缘设备和其他硬件平台上高效运行；
    lead design，implement and optimize AI algorithm to
    run efficiently on edge device and other hardware platform；

  • 利用分析工具和性能分析来确定模型优化的领域；
    utilize profiling tool and performance analysis
    to identify area for model optimization；

2.2. Framework and Inference Engine

  • Framework and Inference Engine：框架和推理引擎

  • 与TF、PyTorch和ONNX等AI框架合作开发和优化模型；
    work with AI framework such as TensorFlow，
    PyTorch and ONNX to develop and optimize model；

  • 利用OpenVino、QNN/SNPE、RyZen AI、DirectML和TensorRT等推理工具包实现高性能执行；
    utilize inference toolkit like OpenVino，QNN/SNPE，RyZen AI，
    DirectML and TensorRT to achieve high-performance execution；

2.3. Architecture and Design

  • Architecture and Tech Stack Design：架构和技术栈设计

  • 为ML解决方案的数据集成、数据流和数据存储设计和实现体系结构和技术栈；
    design and implement architecture and tech stack for data
    integration，data flow and data storage for ML solution；

  • 确保模型与生产系统的无缝集成；
    ensure seamless integration of model with production system；

2.4. Model Deployment and Integration

  • Model Deployment and Integration：模型部署和集成；

  • 将AI模型部署到生产环境，确保高效可靠地运行；
    deploy AI model to production environment，
    ensuring they operate efficiently and reliably；

  • 将模型与现有生产系统和工作流集成；
    integrate model with existing production system and workflow；

2.5. Research and Innovation

  • Research and Innovation：研究与创新；

  • 及时了解AI技术和部署策略的最新进展，stay current with the latest
    advancement in AI technology and deployment strategy；

2.6. Collaboration and Documentation

  • Collaboration and Documentation：协作和文档；

  • 与包括硬件和软件工程师在内的跨职能团队合作，实现最佳AI模型部署；
    collaborate with cross-functional team，including hardware
    and software engineerto achieve optimal AI model deployment；

  • 记录战略，方法，成果，以促进持续改进；document strategy，methodology
    and outcome to facilitate continuous improvement

3. Required Skill

  • 熟练掌握C、C和Python，Java等语言的编程技能； strong programming skill in language such as C，C，Python，Java；

  • 熟练使用至少一个AI框架，如TF、PyTorch或ONNX；
    skill with at least one AI framework such as TF，PyTorch，ONNX；

  • 经过验证的设计和优化算法以在边缘设备上运行的能力；
    proven ability to design and optimize algorithm to run on edge device；

  • 熟悉其中一个推理引擎，如OpenVino、QNN/SNPE、RyZen AI、DirectML、TensorRT；
    familiarity with one of inference engine such as above；

  • 具有设计和实施数据集成、流、存储和ML解决方案架构的经验；
    experience in designing and implementing architecture
    for data integration，flow，storage and ML solution；

  • 在将模型与生产系统集成方面有着良好的业绩记录；
    proven track record in integrating model with production system；

  • 有很强的解决问题的能力和创新的热情；
    strong problem-solving skill and passion for innovation；

4. Qualification

  • MIS，计算机工程或相关专业；

  • 有生成式AI集成，云AI和边缘AI经验；
    experience with Generative AI integration, Cloud AI and Edge AI；

  • 有使用OpenCL、SYCL或CUDA进行并行编程的经验；
    experience with parallel programming using OpenCL，SYCL or CUDA；

  • 在AI应用程序内存优化方面拥有丰富的经验；
    proven experience in memory optimization for AI application；

  • 了解AI应用程序的端到端运行时技术栈；
    knowledge of end-to-end runtime stack for AI application；

  • 优秀的计算机视觉、自然语言处理、语音/音频处理和深度学习方法的理论和实践知识；
    excellent knowledge of theory and practice of CV，
    NLP, Speech/Audio processing, and DL method；

5. Additional Qualification

  • 有机器学习编译器开发经验；
    experience with ML compiler development；

  • 有硬件设计权衡分析经验
    experience with hardware design trade-off analysis；
