2. 12 range()

  • 默认从0开始，增量为1；

  • range(inclusiveStart, exclusiveEnd, step)

# [0, 1, 2, 3, 4] [2, 3, 4]
print(list(range(5)),list(range(2, 5)))

# [1, 2, 3, 4, 5, 6, 7] []
print(list(range(1, 8)),list(range(5, 3)))
# [2, 6, 10, 14]
print(list(range(2, 15, 4)))

# [11, 10, 9, 8, 7, 6, 5, 4, 3]
print(list(range(11, 2, -1)))

print(list(range(-5, 5)))

5. 15 Break Continue

  • break只退出它所在的最内层循环，
    若有嵌套循环，则需多次使用break才能退出所有循环；

  • continue只影响它所在的最内层循环，若有嵌套循环，
    需多次使用continue来跳过所有循环中的当前迭代；

  • break：完全停止循环，continue：跳过循环的当前迭代并继续下一次迭代，
    break和continue，都可在for和while循环中使用；

5.1. Break

  • exit or break out of loop；

  • want to stop loop and move on to next code section；

result = []
for i in range(10):
    if i == 5:
        break
    result.append(i)
# [0, 1, 2, 3, 4]
print(result)

5.2. Continue

  • skip current iteration without stopping loop entirely；

  • continue to next iteration without
    executing rest code in current iteration；

result = []
for i in range(10):
    if i % 2 == 0:
        continue
    result.append(i)
# [1, 3, 5, 7, 9]
print(result)

6. 16 For While Else

  • while和for循环可与else合用，else子句允许指定将在循环完成执行后要执行的代码，
    如当想在循环完成运行后检查某些条件，并根据结果采取适当的操作时，很有用；

6.1. For Else

for i in range(5):
    print(i)
else:
    print('executing loop complete...')

6.2. While Else

i = 0
while i < 5:
    print(i)
    i += 1
else:
    print('executing loop complete...')

6.3. Else Break

  • else块只有在循环正常完成执行时才会执行，
    若用break语句提前退出循环，则不会执行else块    ；

 
for i in range(5):
    if i == 2:
        break
    print(i)
else:
    # 0 1
    print('executing loop with break...')

for i in range(5):
    if i == 6:
        break
    print(i)
else:
    # 0 1 2 3 4 executing loop without break...
    print('executing loop without break...')

7. 17 Str Casting Method

  • str.upper()：全大写，str.lower()：全小写；

  • str.swapcase()：大小写互转；

  • str.title()：大写单词中的每个首字母；

  • str.capitalize()：大写句子中的首字母；

Str = 'Hello World'

# HELLO WORLD hello world hELLO wORLD
print(Str.upper(),Str.lower(),Str.swapcase())

Str = 'hello world'

# Hello World Hello world
print(Str.title(),Str.capitalize())

8. 18 Str Modifying

8.1. Replace

  • str.replace(old,new)；

greeting = 'Hello World'

# Hello Universe
print(greeting.replace('World', 'Universe'))

# ABaABAB
print('abaabab'.replace('ab', 'AB'))

8.2. Strip

  • str.strip()：删除字符串开头或结尾的任何字符，
    包括空格，制表符，换行符，且可删除任何指定字符序列；

greeting = '      Hello World!   '

# Hello World!
print(greeting.strip())

# abaabab
print('  abaabab'.strip())

# hello
print('__hello__'.strip('_'))
#   _hey hey
print('  _hey hey___'.strip('_'))

8.3. Lstrip

  • str.lstrip()，类strip()，移除左边开始的whitespace或指定参数；

greeting = '      Hello World!   '
# [Hello World!   ]
print(greeting.lstrip())

# Hey Universe
print('Hey Universe'.lstrip())

# abaabab
print('  abaabab'.lstrip())

# hello__
print('__hello__'.lstrip('_'))

#   _hey hey___
print('  _hey hey___'.lstrip('_'))

8.4. Rstrip

  • str.rstrip()，类lstrip()；

greeting = '      Hello World!   '
#       Hello World!
print(greeting.rstrip())

# Hey Universe
print('Hey Universe'.rstrip())

#   abaabab
print('  abaabab'.rstrip())

# __hello
print('__hello__'.rstrip('_'))

#   _hey hey
print('  _hey hey___'.rstrip('_'))

8.5. Removeprefix Removesuffix

  • str.removeprefix()，str.removesuffix()；

greeting = '      Hello World!   '

# [     Hello World!   ]
print(greeting.removeprefix(' '))

# ey Universe
print('Hey Universe'.removeprefix('H'))

# baabab
print('  abaabab'.removeprefix('  a'))

# _hello__
print('__hello__'.removeprefix('_'))

# [  _hey hey___]
print('  _hey hey___'.rstrip(' '))

9. 19 Split Join Str

9.1. Splitting

  • split默认使用whitespace分割；

  • 使用regular expression；

sentence = 'Hello World!'
word_list = sentence.split(' ')
# ['Hello', 'World!']
print(word_list)

msg = '''Hello my dear friend,
How is life?
'''

# ['Hello', 'my', 'dear', 'friend,', 'How', 'is', 'life?']
print(msg.split())
import re

sentence = 'Hello, Elf! How are you?'

# 根据逗号(comma)，空格(space)，感叹号(exclamation mark)，问号(question mark)分割
word_list = re.split(r'[, !?]', sentence)

# ['Hello', '', 'Elf', '', 'How', 'are', 'you', '']
print(word_list)

9.2. Joining

word_list = ['Hello', 'Elf!', 'How', 'are', 'you', '?']
sentence = ' '.join(word_list)

# Hello Elf! How are you ?
print(sentence)

sentence = 'Hi Elf! How are you doing?'
word_list = sentence.split('!')

word_list[0] = 'Hi Python'
new_sentence = ' -- '.join(word_list)

# Hi Python --  How are you doing?
print(new_sentence)

10. 20 List Method

10.1. Append

  • list.append(item)；

num_list = [1, 2, 3];
num_list.append(4)
num_list ☞ [1, 2, 3, 4]

10.2. Extend

  • list.extend(iterable)，末尾添加；

  • iterable：如list或tuple；

num_list = [1, 2, 3]
num_list.extend([4, 5, 6])

num_list ☞ [1, 2, 3, 4, 5, 6]

10.3. Insert

  • list.insert(index, item)；

  • 索引相对应的元素不会被删除或修改，而是往后移；

num_list = [1, 2, 3]
num_list.insert(1, 4)
num_list ☞ [1, 4, 2, 3]

10.4. Remove

  • list.remove(item)；

num_list = [1, 2, 3, 4, 4]
num_list.remove(4)
num_list ☞ [1, 2, 3, 4]

10.5. Pop

  • list.pop(index)；

  • 若未指定索引，默认remove最后一个元素；

num_list = [1, 2, 3]
num_list.pop(1)
num_list ☞ [1, 3]

10.6. Index

  • list.index(item)，首次匹配元素值的索引；

num_list = [1, 2, 3, 4, 4]

num_list.index(4)

index ☞ 3

10.7. Count

  • list.count(item)；

  • 统计特殊元素出现的次数

num_list = [1, 2, 3, 4, 4]
num_list.count(4)
count ☞ 2

10.8. Sort

  • list.sort()，默认升序；

num_list = [3, 1, 4, 2]
num_list.sort()
num_list ☞ [1, 2, 3, 4]

10.9. Reverse

  • list.reverse()；

num_list = [1, 2, 3]
num_list.reverse()
num_list ☞ [3, 2, 1]

10.10. Clear

  • list.clear()，删除所有元素；

  • remove：effectively emptying，也可使用del关键字；

  • del关键字用于从内存delete列表的variable entirely，
    而clear仅empty(清空) list；

num_list = [1, 2, 3]
num_list.clear()
num_list ☞ []
num_list = [1, 2, 3]
del num_list[1]
# [1, 3]
print(num_list)

# del num_list
# 将会抛错，因num_list变量将不存在；
# print(num_list)

num_list = [1, 2, 3]
num_list.clear()
# []
print(num_list)
