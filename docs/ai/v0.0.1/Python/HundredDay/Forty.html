1. 31 Enumerate Zip

1.1. Enumerate

  • enumerate函数向可迭代对象添加一个计数器，并以枚举对象的形式返回它；

  • 此对象可转换为元组列表，该列表包含可迭代项中的索引和相应项；

fruit_list = ['apple', 'banana', 'cherry']
for i, fruit in enumerate(fruit_list):
    print(i, fruit)

for i, fruit in enumerate(fruit_list,1):
    print(i, fruit)

# 索引默认从零开始
"""
0 apple
1 banana
2 cherry

1 apple
2 banana
3 cherry
"""

1.2. Zip

  • zip函数：将两个或多个可迭代项组合为单个可迭代项，
    通过从每个可迭代项中获取相应的项，形成最终可迭代项；

fruit_list = ['apple', 'banana', 'cherry']
color_list = ['red', 'yellow', 'red']
for fruit, color in zip(fruit_list, color_list):
    print(fruit, color)

fruit_list = ['apple', 'banana', 'cherry']
color_list = ['red', 'yellow']
for fruit, color in zip(fruit_list, color_list):
    print(fruit, color)

# 注：得到的可迭代的长度等于最短输入可迭代的的长度

"""
apple red
banana yellow
cherry red

apple red
banana yellow
"""

2. 32 Lambda Function

  • 用lambda关键字创建匿名函数

2.1. Lambda Syntax

lambda arguments: expression
add = lambda x, y: x + y
result = add(5, 3)

# 8
print(result)

2.2. Map

  • map函数#：以一个函数和一个可迭代函数作为参数，并返回一个新的可迭代函数；

  • 该函数应用于每个元素，如要用lambda函数和映射对数字列表中的所有元素进行平方；

num_list = [1, 2, 3, 4, 5]
square = list(map(lambda x: x**2, num_list))

# [1, 4, 9, 16, 25]
print(square)

2.3. Filter

  • filter函数#：将一个函数和一个可迭代函数作为参数，并返回一个新的可迭代函数，

  • 其中仅包含函数返回True的元素，如用lambda函数和过滤器从数字列表中获取偶数列表；

num_list = [1, 2, 3, 4, 5]
even = list(filter(lambda x: x % 2 == 0, num_list))

# [2, 4]
print(even)

2.4. Reduce

  • reduce函数#：functools模块的内置函数，以函数和可迭代项作为参数，
    并返回一个值，该值是将函数累积cumulatively应用于可迭代项的元素的结果；

  • 如用lambda函数和reduce找到数字列表中所有元素的乘积；

from functools import reduce

num_list = [1, 2, 3, 4, 5]
product = reduce(lambda x, y: x * y, num_list)

# 120
print(product)

2.5. Lambda Limitation

  • 限制一：只能包含一个表达式，即不能包含语句或多行代码，
    另表达式必须是Python中的有效表达式，且必须返回一个值；

  • 限制二：lambda函数不能包含任何注释或类型提示；
