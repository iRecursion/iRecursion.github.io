1. 算术运算符

  • A：加(+)，减(-)，乘(*)，除(/)；

  • B：取整(//)：9 // 2 ☞ 4；

  • C：取余或取模(%)：9 % 2 ☞ 1；

  • D：幂：次方，乘方，两个*，2 ** 3 ☞ 8；

  • E：优先级：先幂，其次乘除，取整，取模，最后加减，从左向右算；

  • F：小技巧：将减法转换为加法，除法转换为乘法后再进行计算；

2. 赋值运算符

  • =，+=，-=，*=，/=，//=(取整)，%=(取模，取余)，⭐⭐=(幂)；

  • 所有赋值运算符优先级相同，赋值运算符优先级很低，仅高于(成员，身份，逻辑)运算符；

3. 关系/比较 运算符

  • ==，!=，>=，⇐，>，<；

4. 位运算符

运算符 名称 描述

&

按位与

对应位都为1则返回1，否则返回0

|

按位或

对应位有一个为1则返回1，否则返回0

^

按位异或

对应位不同时，则返回1，否则返回0

~

按位取反

对数据的每个二进制位取反操作，0变1，1变0

<<

左移

相当于乘以2的n次方，高位丢弃，低位补零

>>

右移

相当于除以2的n次方，低位丢弃，高位补0

5. 逻辑运算符

  • 逻辑与：and，a and b;逻辑或：or， a or b；逻辑非：not， not a；

  • 短路计算：a and b：若a为假，则无需计算b，a or b，若a为真，则无需计算b；

  • 优先级：not ☞ and ☞ or；

6. 成员运算符

运算符 描述 示例

in

若在指定的序列中找到值返回True，否则返回False

3 in (1,2,3) 返回True

not in

若在指定的序列中没找到值返回True，否则返回False

3 not in (1,2,3) 返回False

  • 主要应用在序列中：

    • x in seq，x not in seq；
      x表示某数据，seq表示集合；

  • 判断子串：print("wo" in "hello world")

7. 身份运算符

运算符 描述

is

判断两标识符是否引用同一对象，

is not

判断两标识符是否不是引用同一对象，True：是，False：否

 
obj1,obj2,obj3 = "elf","pal","elf"

# False True True
print(obj1 is obj2,obj1 is not obj2,obj1 is obj3)

  • 注：is和==区别：

    • is：判断两个变量引用实体是否为同一Id，id()函数获取实体的Id(地址)；

    • ==：判断两变量的值是否相同，但Id不一定相同

 
num1,num2 = 1,1

# 140730277435832 140730277435832 True True
print(id(num1),id(num2),num1 == num2,num1 is num2)

num3,num4 = 123456,123456
# 2013827280432 2013827280432 True True
print(id(num3),id(num4),num3 == num4,num3 is num4)

8. 优先级

  • A：**(幂)；

  • B：~，+，-：按位取反，一元运算符(正负号)；

  • C：*，/，%，//：乘，除，取余(取模)，取整；

  • D：+，-：加法，减法；

  • E：<<，>>：左移，右移；

  • F：&：按位与；

  • G：：^；

  • H：>，<，>=，⇐：比较运算符；

  • I：==，!=：等于运算符；

  • J：=，+=，-=，*=，/=，%=，//=，⭐⭐=：赋值运算符；

  • K：is，is not；

  • L：in，not in；

  • M：not，or，and：逻辑运算符；
