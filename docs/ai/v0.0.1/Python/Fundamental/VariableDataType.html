1. 变量

  • 变量名 = 变量值

  • 多变量相同：v1 = v2 = v3 = 5

  • 多变量值不同，元组赋值：v4,v5 = 10,20

  • 一般相关变量会一起定义，如坐标等；

v1 = v2 = v3 = 5

v4,v5 = 10,20
print(v1,v2,v3,v4,v5)

  • 变量修改，可修改类型

year = 2023
print(year)	# 2023

year = 2024
print(year)	# 2024

2. 常量

  • 常量值永不发生改变，约定大写表示常量；

3. 变量命名

  • 标识符：可由字母，下划线和数字组成，不能以数字开头，不能与关键字重名；

  • Python采用蛇形命名法(snake_case)，单词都小写，之间用下划线连接；
    https://zh.wikipedia.org/wiki/%E8%9B%87%E5%BD%A2%E5%91%BD%E5%90%8D%E6%B3%95

  • 小驼峰式命名法(lowerCamelCase)，大驼峰式命名法(UpperCamelCase)

5. 浮点型

  • 浮点型运算可能有四舍五入的误差；

n1 = 0.1
n2 = 0.2

# 0.30000000000000004
print(n1 + n2)

n1 = 2.1
n2 = 15.256

# 四舍五入：round(number, ndigits)
n3 = round(n1 + n2,2)
# 四舍五入： 17.36
print("四舍五入：",n3)

import math
# 向上取整：ceil
n4 = math.ceil(n1 + n2)

# 向上取整： 18
print("向上取整：",n4)

# 向下取整：floor
n5 = math.floor(n1 + n2)
# 向下取整： 17
print("向下取整：",n5)

6. 布尔型

  • 判断真假，可选值：True(1)，False(0)；

  • Python可解释为假的值：None，0，0.0，False，
    所有空容器(空列表，空元组，空字典，空集合，空字符串)；

7. 字符串

  • 可用双引号或单引号''，""，
    或三引号(''' '''，""" """)表示多行字符串，类似于Java：

  • 运算：

    • 字符串加法：字符串拼接；

    • 字符串乘法：重复字符串；

name = "elf"
age = 18
# elf18 elfelfelf
print(name + str(age),name * 3)

  • 索引：顺序从零开始，最后一个从-1开始

s = "elf,hello world"
# 切片：半闭半开：f elf r
print(s[2],s[0:3],s[-3])

# 切片：变量名[起始索引:结束索引 + 1:步数]
sn = "123456789"

# 从零开始到第九各位置，每次取两个字符
# 其实位置默认为0，结束索引默认-1，步数默认为1
# 13579 147
print(sn[0:9:2],sn[::3])

# 字符串反转
print(sn[-1:-10:-1],sn[::-1])

s1,s2 = "10","1a"

9. 小整数地址

  • Python中的小整数：-5 ~ 256，类似于Java中IntegerCache；

  • 当在Python创建整数对象，会根据整数的值动态分配内存空间，
    对小整数会采取"小整数缓存"机制来优化内存使用，即这类整数对象会被重用；

  • Python提供内置函数id()来获取对象的唯一标识符，通常可近似的
    表示对象的内存地址，油解释器生成，并非真实的内存地址；

 
name = "elf"
print(id(name))

na = 8
print(id(na))

nb = 8
print(id(nb))
