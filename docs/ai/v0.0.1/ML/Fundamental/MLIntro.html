1. Definition

  • 让机器学规则，学的规则并非用人教，而是把input，output丢给它学习；

  • 没有人教它怎么做这些事情，但它有资料；

  • 视其所以，观其所由，察其所安，人焉廋哉？近朱者赤，近墨者黑；

  • study field give computer ability to learn without being explicitly programmed；

  • computational method use existing data to make prediction

2. ML Paradigm

2.1. Supervised Learning

  • each input has output，available immediately or with delay of fixed duration

  • two type：classification and regression

  • example：weather prediction，face recognition

2.2. Reinforcement Learning

  • output available but with delay of variable duration

  • Example：Chess Playing，Control System，Path Planning

2.3. Unsupervised Learning

  • no output part

  • Example：KMean Clustering，Principal Component Analysis

3. Polynomial Fitting

12. UCI Dataset

12.1. Iris

  • Goal：predict iris type in Hawaii；

  • Dataset Specification：

    • 150 instance，3 class，4 attribute(feature)；

    • sepal length，sepal width，petal length，petal width；

12.2. Wine

  • Goal：determine wine origin via 13 chemical constituent；

  • Dataset Specification：178 instance，3 class，13 attribute；

12.3. Abalone

  • Goal：predict abalone age；

  • Dataset Specification：

    • 4177 instance，29 class，8 attribute(feature)；

    • sex，length，diameter，height，whole weight，
      shucked weight，viscera weight，shell weight；

    • 1 output：ring(+1.5 give age in years)

12.4. Mushroom Classification

  • Goal：determine mushroom poisonous or edible；

  • Dataset Specification：8124 instance，2 class，22 attribute；

12.5. Liver Disorder

  • Goal：use variable from blood test and
    alcohol consumption to see if liver disorder exist；

  • Dataset Specification：

    • 345 instance，2 class，6 attribute；

    • first five are result from blood test；

    • last one is alcohol consumption per day；

12.6. Credit Screening

  • Goal：determine people who grant credit

  • Dataset Specification：125 instance，2 class，15 attribute

12.7. House Price Prediction

  • Goal：predict house price near Boston

  • Dataset Specification：506 instance，13 attribute
