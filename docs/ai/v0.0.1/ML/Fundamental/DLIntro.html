1. Basic Concept

  • 模型：包含大量未知参数的函数

  • 训练：通过大量的数据去迭代逼近这些未知参数的最优解

  • ML：研究 计算机怎样模拟或实现人类的学习行为，以获取新的知识或技能，
    重新组织已有的知识结构使之不断改善自身的性能的学科    ；即：从样本中学习的智能程序；

  • DL：源于人工神经网络的研究，其动机在于建立、模拟人脑进行分析学习的神经网络，
    它模仿人脑的机制来解释数据，如图像，声音和文本    ；

  • 不论ML或DL，都是通过对大量数据的学习，掌握数据背后的分布规律，
    进而对符合该分布的其他数据进行准确预测；

2. Classification Regression

  • ML两种类型任务：分类任务Classification和回归任务Regression

  • 分类任务：对离散值预测，根据每个样本的值/特征预测该样本属于类型A，B，C；
    如情感分类、内容审核，相当于学习一个分类边界(决策边界)，用分类边界把不同类别的数据区分开；

  • 回归任务：对连续值预测，根据每个样本的值/特征预测该样本的具体数值，
    如房价预测，股票预测等，相当于学习到这一组数据背后的分布，能根据数据的输入预测该数据的取值；

  • 分类与回归根本区别：输出空间是否为一个 度量空间；

  • f(x) → y,x ∈ A,y ∈ B

  • 分类问题：目的：寻找决策边界，其输出空间B不是度量空间，即定性，categorical；

  • 回归问题：目的：寻找最优拟合，其输出空间B是一个度量空间，即定量，numerical；
    通过度量空间衡量预测值与真实值间的"误差大小"，当真实值为10，预测值为5时，误差为5，预测值为8时，误差为2；
