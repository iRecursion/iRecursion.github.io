1. Basic Concept

1.1. Human Perspective

1.2. Math Perspective

  • 模型：包含大量未知参数的函数

  • 训练：通过大量的数据去迭代逼近这些未知参数的最优解

1.3. Fundamental Concept

  • ML：研究 计算机怎样模拟或实现人类的学习行为，以获取新的知识或技能，
    重新组织已有的知识结构使之不断改善自身的性能的学科    ；即：从样本中学习的智能程序；

  • DL：源于人工神经网络的研究，其动机在于建立、模拟人脑进行分析学习的神经网络，
    它模仿人脑的机制来解释数据，如图像，声音和文本    ；

  • 不论ML或DL，都是通过对大量数据的学习，掌握数据背后的分布规律，
    进而对符合该分布的其他数据进行准确预测；

2. ML Task Type

2.1. Classification Regression

  • ML两种类型任务：分类任务Classification和回归任务Regression；

  • 分类任务：对离散值预测，根据每个样本的值/特征预测该样本属于类型A，B，C；
    如情感分类、内容审核，相当于学习一个分类边界(决策边界)，用分类边界把不同类别的数据区分开；

  • 回归任务：对连续值预测，根据每个样本的值/特征预测该样本的具体数值，
    如房价预测，股票预测等，相当于学习到这一组数据背后的分布，能根据数据的输入预测该数据的取值；

  • 分类与回归根本区别：输出空间是否为一个 度量空间；

  • f(x) → y,x ∈ A,y ∈ B

  • 分类问题：目的：寻找决策边界，其输出空间B不是度量空间，即定性，categorical；

  • 回归问题：目的：寻找最优拟合，其输出空间B是一个度量空间，即定量，numerical；
    通过度量空间衡量预测值与真实值间的"误差大小"，当真实值为10，预测值为5时，误差为5，预测值为8时，误差为2；

3. ML Category

3.1. Supervised Learning

  • 有监督学习：利用大量的标注数据来训练模型，对模型的预测值和数据的真实标签计算损失，
    然后将误差进行反向传播(计算梯度、更新参数)，通过不断的学习，最终可获得识别新样本的能力；

  • 每条数据都有正确答案，通过模型预测结果与正确答案的误差不断优化模型参数；

3.2. Unsupervised Learning

  • 无监督学习：不依赖任何标签值，通过对数据内在特征的挖掘，
    找到样本间的关系，如聚类相关的任务；

  • 只有数据没有答案，常见的是聚类算法，通过衡量样本之间的距离来划分类别；

  • 有监督和无监督最主要区别：模型在训练时是否需人工标注的标签信息；

3.3. Semi Supervised Learning

  • 半监督学习：用有标签数据和无标签数据来训练模型

  • 一般假设无标签数据远多于有标签数据，如使用有标签数据训练模型，
    然后对无标签数据分类，再使用正确分类的无标签数据训练模型；

  • 利用大量的无标注数据和少量有标注数据进行模型训练；

3.4. Self Supervised Learning

  • 自监督学习：ML的标注数据源于数据本身，而非由人工标注

  • 目前主流大模型的预训练过程都采用自监督学习，
    将数据构建成完型填空形式，让模型预测对应内容，实现自监督学习；

  • 通过对数据处理，让数据的一部分成为标签，由此构成大规模数据进行模型训练；

3.5. Distant Supervision Learning

  • 远程监督学习：主要用于关系抽取任务，采用bootstrap的思想通过
    已知三元组在文本中寻找共现句，自动构成有标签数据，进行有监督学习    ；

  • 基于现有的三元组收集训练数据，进行有监督学习

  • 三元组：中国，首都，北京，三元组类似于图数据库的关系，
    中国是主实体，首都是关系，北京是相关实体；

3.6. Reinforcement Learning

  • 强化学习：智能体根据已有的经验，采取系统或随机的方式，去尝试各种可能答案的方式学习，
    且智能体会 通过环境反馈的奖赏来决定下一步的行为，并为获得更好的奖赏来进一步强化学习；

  • 以获取更高的环境奖励为目标优化模型；
