1. Cross Cluster Mirroring

  • жҹҗдәӣеә”з”ЁеҸҜиғҪйңҖжқҘиҮӘеӨҡдёӘж•°жҚ®дёӯеҝғзҡ„ж•°жҚ®пјҢз®ЎзҗҶе‘ҳе°ҶйңҖеңЁйӣҶзҫӨй—ҙиҝһз»ӯеӨҚеҲ¶(copy)ж•°жҚ®пјӣ

  • mirroring applicationзҡ„ж ёеҝғжҳҜж¶Ҳиҙ№иҖ…е’Ңз”ҹдә§иҖ…пјҢеңЁй•ңеғҸиҝҮзЁӢ
    (mirroring process)дёӯпјҢе®ғдјҡж¶Ҳиҙ№AйӣҶзҫӨдёӯзҡ„ж•°жҚ®пјҢе№¶е°Ҷе…¶еҸ‘еёғеҲ°BйӣҶзҫӨпјӣ

  • жү§иЎҢmirroringзҡ„е·Ҙе…·пјҡMirror Maker 2е’ҢuReplicatorпјӣ

2. Active Active Architecture

  • еҪ“дёӨдёӘжҲ–еӨҡдёӘж•°жҚ®дёӯеҝғе…ұдә«йғЁеҲҶжҲ–е…ЁйғЁж•°жҚ®пјҢдё”жҜҸдёӘж•°жҚ®дёӯеҝғ
    йғҪиғҪproduceе’Ңconsume eventж—¶пјҢе°ҶдҪҝз”ЁжӯӨжһ¶жһ„пјӣ

  • з”ҹдә§иҖ…еңЁдёӨдёӘйӣҶзҫӨдёӯз”ҹжҲҗзӣёеҗҢзҡ„дё»йўҳпјҢдё”еңЁйӣҶзҫӨдёӯзҡ„дёӨдёӘдё»йўҳй—ҙиҝӣиЎҢй•ңеғҸпјӣ

2.1. Advantage

  • иғҪд»Һйҷ„иҝ‘зҡ„ж•°жҚ®дёӯеҝғдёәз”ЁжҲ·жҸҗдҫӣжңҚеҠЎпјҢйҖҡеёёе…·жңүжҖ§иғҪдјҳеҠҝпјӣ

  • еҶ—дҪҷе’Ңеј№жҖ§пјҡRedundancy and ResilienceпјҢеӣ жҜҸдёӘж•°жҚ®дёӯеҝғ
    йғҪе…·жңүжүҖжңүеҠҹиғҪпјҢж•…жҹҗж•°жҚ®дёӯеҝғдёҚеҸҜз”ЁпјҢиҜҫе°Ҷз”ЁжҲ·еј•еҜјеҲ°е…¶е®ғж•°жҚ®дёӯеҝғпјӣ

2.2. Disadvantage

  • дё»иҰҒзјәзӮ№(drawback)пјҡеңЁеӨҡдёӘдҪҚзҪ®ејӮжӯҘиҜ»еҸ–е’Ңжӣҙж–°ж•°жҚ®ж—¶пјҢеңЁйҒҝе…ҚеҶІзӘҒж–№йқўеӯҳеңЁжҢ‘жҲҳпјӣ

  • еҸҰйңҖдҪҝз”Ёж¶Ҳиҙ№иҖ…(йҖҡеёёдҪҝз”Ёж—¶й—ҙжҲі)еӨ„зҗҶд»ҺдҪ•еӨ„иҜ»еҸ–пјҢеӣ еҒҸз§»йҮҸдёҚдёҖе®ҡеңЁйӣҶзҫӨй—ҙеҗҢжӯҘпјӣ

3. Active Passive Architecture

  • еңЁжҹҗдәӣжғ…еҶөдёӢпјҢеҜ№еӨҡдёӘйӣҶзҫӨзҡ„е”ҜдёҖиҰҒжұӮжҳҜж”ҜжҢҒжҹҗз§ҚзҒҫйҡҫеңәжҷҜпјҢ
    жҲ–йҖҡиҝҮй•ңеғҸж•ҙдёӘйӣҶзҫӨжқҘе®һзҺ°жӣҙеҝ«зҡ„жң¬ең°иҜ»еҸ–пјӣ

  • д№ҹи®ёеңЁеҗҢдёҖдёӘж•°жҚ®дёӯеҝғдёӯжңүдёӨдёӘйӣҶзҫӨпјҢеҜ№жүҖжңүеә”з”ЁзЁӢеәҸдҪҝз”ЁдёҖдёӘйӣҶзҫӨпјҢдҪҶеёҢжңӣз¬¬дәҢдёӘйӣҶзҫӨ
    еҢ…еҗ«(еҮ д№Һ)еҺҹе§ӢйӣҶзҫӨдёӯзҡ„жүҖжңүдәӢд»¶пјҢиӢҘеҺҹе§ӢйӣҶзҫӨе®Ңе…ЁдёҚеҸҜз”ЁпјҢеҲҷеҸҜдҪҝз”ЁиҝҷдәӣдәӢд»¶пјӣ

  • жӯӨжһ¶жһ„дёӯпјҢз”ҹдә§иҖ…д»…е°Ҷж•°жҚ®еҸ‘еёғеҲ°жҙ»еҠЁйӣҶзҫӨпјҢиў«еҠЁйӣҶзҫӨдёҚжҺҘж”¶д»»дҪ•еҶҷе…ҘпјҢ
    е®ғеҸӘжҳҜйҖҡиҝҮMirror Makerд»Һдё»еҠЁйӣҶзҫӨжҺҘж”¶й•ңеғҸж•°жҚ®пјӣ

3.1. Advantage

  • ж— йңҖжӢ…еҝғи®ҝй—®ж•°жҚ®гҖҒеӨ„зҗҶеҶІзӘҒе’Ңе…¶е®ғжһ¶жһ„еӨҚжқӮжҖ§(architectural complexity)пјӣ
