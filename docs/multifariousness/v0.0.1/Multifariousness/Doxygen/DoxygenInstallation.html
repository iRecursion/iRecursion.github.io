1. 概览

  • Doxygen采用自安装归档形式；

  • 类似于PlantUml，推荐安装GraphViz，参考配置文件中的HAVE_DOT选项；
    https://www.doxygen.nl/manual/config.html#cfg_have_dot

  • 若生成Pdf输出或使用数学公式，还需安装LaTeX和Ghostscript；

  • LaTeX存在多种发行版，推荐与Doxygen配合使用的流行产品MikTex和proTeXt(retired)；

  • 安装完LaTeX和Ghostscript后，需确保latex.exe，pdflatex.exe，
    gswin32c.exe(或gswin64c.exe)存在于环境变量或PATH中；

  • https://graphviz.org

  • https://miktex.org
    https://www.tug.org/protext
    https://en.wikipedia.org/wiki/LaTeX

  • https://www.ghostscript.com
    https://en.wikipedia.org/wiki/Ghostscript
