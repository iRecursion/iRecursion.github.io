1. 概览

  • rustup安装Rust，将rustc、cargo、rustup等安装到Cargo的bin目录；

  • Unix位置：$HOME/.cargo/bin；

  • Win位置：%USERPROFILE%\.cargo\bin；

  • CARGO_HOME：包含Cargo使用的缓存文件

  • RUSTUP_HOME：rustup的root文件夹，存储已安装的工具链和配置项

2. ABI Type

MSVC GNU

VS使用的native(MSVC) ABI，默认ABI

GCC toolchain使用的GNU ABI

SVC版Rust与VS生成的软件互操作

GNU版Rust与MinGW/MSYS2工具链构建的GNU软件互操作

MSVC ABI：Rust需安装VS

GNU ABI：基本使用无需依赖，建议安装带有MinGW的完整MSYS2

3. Obtain Artifact

  • Target Triple：

    • x86_64-pc-windows-gnu：64位MinGW

    • x86_64-pc-windows-msvc：64位MSVC

  • Rust Version

    • 1.81.0，1.80.0，1.79.0，1.78.0，1.77.0，1.76.0

  • Rustup Version

    • 1.27.1，1.27.0，1.26.0，1.25.0，1.24.0

  • Executable Installer：

    • https://static.rust-lang.org/dist/rust-$rust-version-$target-triple.msi

    • https://static.rust-lang.org/dist/rust-1.81.0-x86_64-pc-windows-gnu.msi

    • https://static.rust-lang.org/rustup/archive/$rustup-version/$target-triple/rustup-init.exe

    • https://static.rust-lang.org/rustup/archive/1.27.1/x86_64-pc-windows-gnu/rustup-init.exe

  • 不推荐使用msi独立安装包，功能缺失，也不推荐使用官网脚本方式安装，
    未自定义，不够灵活，推荐使用msvc或gnu方式，此处采用gnu方式，更简洁；

4. Msvc Prerequisite

  • Manual Install：选择VS Build Tool，而非Visual Studio(重量级)；

  • Build Tools for Visual Studio 2022，
    https://visualstudio.microsoft.com/downloads
