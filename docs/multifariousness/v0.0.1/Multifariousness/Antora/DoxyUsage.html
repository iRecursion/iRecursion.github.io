1. Doxywizard Usage

  • Doxywizard：配置和运行Doxygen的GUI前端；

  • 启动Doxywizard时，可指定现有配置文件作为参数，也可使用默认设定；

  • 不推荐使用Wizard Tab，而建议在通读配置Doxyfile文件后，
    使用Expert Tab模式，然后运行Run Tab，来回迭代修改与调试；
Entry Memo

Select/File ☞ Open

指定工作目录，Specify Working Directory

Wizard Tab

快速配置重要设置，其余选项保留为默认值

Expert Tab

所有配置选项，可及时调整

Run Tab

运行Doxygen并查看使用的设置，保存配置等

File ☞ Save

配置好后，可保存配置，以待后续重复使用

2. Html Output

  • Html输出：生成默认的Html Header，Html Footer，Html Style Sheet；

  • doxygen -w html header.html footer.html stylesheet.css <config_file>

  • config_file可选，省略时，Doxygen将搜索名为
    Doxyfile的文件并process，若没找到，则使用默认设置；

3. LaTeX Output

  • 可使用命令生成refman.tex的LATEX_HEADER和LATEX_FOOTER，
    及Header包含的style sheet，通常称为通常为doxygen.sty；

  • doxygen -w latex header.tex footer.tex doxygen.sty <config_file>

4. Emoji Support

  • https://api.github.com/emojis
    https://www.doxygen.nl/dl/github_emojis.zip

  • \emoji 创建表情符号：😄；

  • \emoji smile 或 \emoji :smile:

  • @emoji smile 或 @emoji :smile:

  • Doxygen配置参数LATEX_EMOJI_DIRECTORY，以选择所请求的目录；

  • 查看支持的emoji列表：doxygen -f emoji <outputFileName>；

  • 否则将显示纯表情符号文本，即 :<text>:
