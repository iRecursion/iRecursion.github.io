1. Add User

  • win + r ⇒ netplwiz ⇒ 添加
    ⇒ 不使用Microsoft账户登录(不推荐) ⇒ 本地帐户

2. Show Second

  • 设置 ⇒ 个性化 ⇒ 任务栏 ⇒ 任务栏行为 ⇒ 在系统托盘时钟中显示秒数

3. Relocation

  • C:\Users\Administrator\Desktop

  • 右键 → 属性 → 属性 → 位置 → 移动

  • D:\Install\FreeSystem\Desktop

4. Start Menu

  • C:\ProgramData\Microsoft\Windows\Start Menu\Programs

  • C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs

  • PATH：C:\Program Files\Common Files\Oracle\Java\javapath

5. Windows Terminal

  • win + r ⇒ wt.exe

  • 添加新配置文件 ⇒ 从命令提示符复制配置文件

  • 命令行修改为D:\Install\Dev\Git\bin\bash.exe

  • 图标修改为D:\Install\FreeSystem\Ico\GitBash.png
