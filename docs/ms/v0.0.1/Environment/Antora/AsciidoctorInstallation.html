1. 基础安装

gem install asciidoctor

# 卸载旧版本
gem cleanup asciidoctor

asciidoctor --version

asciidoctor -a toc=left fileName.adoc

2. Diagram

gem install asciidoctor-diagram

asciidoctor -r asciidoctor-diagram fileName.adoc

3. Math

  • Win环境下asciidoctor-mathematical无法安装，
    建议使用Asciidoctor Docker镜像或Antora扩展镜像；

 
gem install asciidoctor-mathematical

asciidoctor -r asciidoctor-mathematical fileName.adoc

4. Highlight

  • Antora只支持highlight.js，Asciidoctor请使用Rouge

gem install coderay

gem install rouge

npm install highlight.js -g
:source-highlighter: rouge

:source-highlighter: highlight.js
