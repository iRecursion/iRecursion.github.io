1. 引言

  • Visual Studio或Microsoft C++ Build Tool太大，
    选择Win11 + MinGW-w64安装Rust；

  • 运行rustup-init.exe安装Rustup，安装Rustup会自动安装Cargo，
    Cargo：Rust的构建工具和包管理器；

3. 安装Mingw64

Type Memo

架构

x86_64：压缩包适于64位的x86处理器架构

版本

13.2.0-release：正式发布的13.2.0版本

操作系统平台

win32：32位Win系统；posix：Linux、Unix等POSIX系统

异常处理

采用结构化异常处理seh：Structured Exception Handling

运行时库

  • MSVCRT(早期)，UCRT(较新)版本的Visual C++

  • ucrt：通用C运行时库(UCRT)

  • msvcrt：Microsoft Visual C++运行时库(MSVCRT)

修订版本

Revision，Rev

  • https://github.com/niXman/mingw-builds-binaries/releases
    x86_64-13.2.0-release-win32-seh-ucrt-rt_v11-rev0.7z

  • 下载解压后设置环境变量：PATH：D:\Install\Green\Mingw64\bin

  • gcc --version && gcc -v

5. 安装步骤

  • 双击运行rustup-init.exe；

  • 3) Don’t Install Prerequisite，Targeting GNU ABI；

  • 2) Customize Installation；

  • Default host triple? [x86_64-pc-windows-msvc]
    x86_64-pc-windows-gnu；

  • Default toolchain? (stable/beta/nightly/none) [stable]
    stable

  • Profile (which tools and data to install)?
    (minimal/default/complete) [default] default；

  • Modify PATH variable? (Y/n) Y；

  • 1) Proceed With Installation (default)

  • Press Enter key to continue.

  • 回车结束安装，rustc -V && cargo -V && rustup -V
