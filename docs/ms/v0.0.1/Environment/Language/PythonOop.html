1. Intro

  • OOP是通过将相关属性和行为绑定到单个对象中来构建程序的方法；

  • 对象像系统的组件，把程序想象成工厂装配线，在装配线的每一步，
    系统组件都会处理一些材料，最终将原料转换为成品；

  • 对象包含数据，如装配线上每个步骤的原材料或预处理材料，
    对象还包括行为，如每个装配线零部件执行的操作；

  • 面向对象编程是编程范式，提供结构化程序的方法，以便将属性和行为绑定到单个对象中；
    如一个对象可代表一个人，其属性包括姓名，年龄，地址，行为可包括行走，说话，呼吸，跑步等；

  • OOP是对具体的、现实世界的事物(如汽车)及事物之间的关系(如公司和员工或学生和教师)进行建模的方法，
    OOP将真实世界的实体建模为软件对象，软件对象具有与它们相关联的数据，且可执行某些操作；
