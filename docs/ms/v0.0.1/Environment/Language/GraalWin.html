1. Graal Win 安装

  • https://www.graalvm.org/latest/docs/getting-started/windows/

JAVA_HOME = D:\Install\Dev\Java\Graal
PATH = %JAVA_HOME%\bin

  • Visual Studio Build Tool和Windows SDK

  • https://visualstudio.microsoft.com/downloads/?q=build+tools
    下载Build Tools for Visual Studio 2022或匹配版本

  • 安装时左边选择Desktop Dev with C++，右边选择Windows SDK；并自定义路径；

  • 查看版本与构建使用：

java --version && native-image --version
1
clean native:compile

gradle nativeCompile

MSVCVARS64=D:\Install\Dev\Mvs\BuildTool\VC\Auxiliary\Build\vcvars64.bat

win + r ⇒ %MSVCVARS64%

native-image HelloWorld
