1. 先决条件

  • 请先参考MySQL镜像：192.168.0.123:3306/elf

  • root - cipher，elf_user - elf_cipher

  • 创建elf数据库，执行相关脚本：
    https://github.com/alibaba/nacos/blob/develop/distribution/conf/mysql-schema.sql

  • 基础定义：https://blog.csdn.net/m0_72673485/article/details/128622922

2. 镜像及配置

docker pull nacos/nacos-server:v2.2.3

mkdir -p /elf/nacos/{log,conf,data}

  • 下载相关配置文件
    https://github.com/alibaba/nacos/blob/develop/distribution/conf/nacos-logback.xml
    https://github.com/alibaba/nacos/blob/develop/distribution/conf/application.properties

容器文件 主机文件

/home/nacos/conf/nacos-logback.xml

/elf/nacos/conf/nacos-logback.xml

/home/nacos/conf/application.properties

/elf/nacos/conf/application.properties

  • 修改配置：vim /conf/application.properties

spring.datasource.platform=mysql
db.num=1
db.url.0=jdbc:mysql://192.168.0.123:3306/elf?characterEncoding=utf8&connectTimeout=1000&socketTimeout=3000&autoReconnect=true&useUnicode=true&useSSL=false&serverTimezone=Asia/Shanghai
db.user.0=root
db.password.0=cipher

3. 授权

  • vim /conf/application.properties

nacos.core.auth.enabled=true

nacos.core.auth.server.identity.key=nacos_user
nacos.core.auth.server.identity.value=nacos_cipher

nacos.core.auth.plugin.nacos.token.secret.key=SecretKey01234567890123456789012345345678999987654901234567890123456789

4. 启动容器

docker run \
--name nacos \
-p 8848:8848 \
-p 9848:9848 \
-p 9849:9849 \
--privileged=true \
-e JVM_XMS=256m \
-e JVM_XMX=1024m \
-e MODE=standalone \
-v /elf/nacos/log:/home/nacos/logs \
-v /elf/nacos/conf:/home/nacos/conf \
-v /elf/nacos/data:/home/nacos/data \
-itd nacos/nacos-server:v2.2.3

5. 访问Nacos

  • docker stop nacos && docker remove nacos

  • http://192.168.0.123:8848/nacos/
    nacos - nacos
