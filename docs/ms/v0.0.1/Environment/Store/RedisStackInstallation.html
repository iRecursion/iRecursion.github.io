1. 相关配置

-v /elf/redis-stack/redis-stack.conf:/redis-stack.conf

-e REDIS_ARGS="--requirepass redis-stack

-e REDIS_ARGS="--save 60 1000 --appendonly yes

-e REDISTIMESERIES_ARGS="RETENTION_POLICY=20"
Entry Memo

Redis Insight端口

8001

Redis Stack Server端口

6379

REDIS_ARGS

Redis额外参数

REDISEARCH_ARGS

RediSearch参数

REDISJSON_ARGS

RedisJSON参数

REDISTIMESERIES_ARGS

RedisTimeSeries参数

REDISBLOOM_ARGS

RedisBloom参数

2. 启动容器

  • -v /elf/soft/stack/redis-stack.conf:/redis-stack.conf

docker run \
--name redis-stack \
-p 6379:16379 -p 18001:8001 \
--privileged=true \
--net network-common \
-v /elf/redis-stack/data/:/data \
-e REDISTIMESERIES_ARGS="RETENTION_POLICY=20" \
-e REDIS_ARGS="--requirepass redis-stack-cipher" \
-itd redis/redis-stack:7.2.0-v8

3. Redis Insight

  • redis/redisinsight:2.46，redislabs/redisinsight:v2

  • http://192.168.0.123:5540
    http://192.168.0.123:5540/api/health/

docker run \
	--name redis-insight \
	-p 5540:5540 \
	--privileged=true \
	--net network-common \
	-v /elf/redis-insight/data/:/data \
	-itd redis/redisinsight:2.46
Variable Default Purpose

RI_APP_PORT

5540

RedisInsight port

RI_APP_HOST

0.0.0.0

RedisInsight host

RI_SERVER_TLS_KEY

private key，file or str in pem format

RI_SERVER_TLS_CERT

public certificate，file or str in pem format

RI_ENCRYPTION_KEY

Redisinsight在本地存储敏感(sensitive)信息，
如DB密码，Workbench历史信息等，使用sqlite3

RI_LOG_LEVEL

info

error，warn，info，http，verbose，debug，silly

RI_FILES_LOGGER

true

日志文件，路径：/data/logs

RI_STDOUT_LOGGER

true

日志到stdout
