1. Preface

  • 说到灵活性，可扩展，有很多方式，spi，热拔插等；

2. 创建模块

  • 创建模块分为@Configuration和@ConditionalOn方式；

  • @Configuration = @ComponentScan + @Bean；

  • @ConditionalOn粒度更细；

2.1. @ComponentScan

@Configuration
@ComponentScan(basePackages = "com.elf.logger")
public class LoggerModuleConfiguration {}

2.2. @Bean

@Configuration
public class LoggerModuleConfiguration {
	@Bean
  	public LoggerService loggerSerivce(){
    	return new Bean();
  	}
}

2.3. @ConditionalOn

  • 若com.elf.logger.enabled设置为false，此模块将完全禁用

@Configuration
@ConditionalOnProperty(name = "com.elf.logger.enabled",
    havingValue = "true", matchIfMissing = true)
public class LoggerModuleConfiguration {
  	//@Bean definition
}

3. 导入模块

3.1. @Import

  • 创建模块后，使用@Import导入到应用，
    这将导入LoggerModuleConfiguration类和附带的所有Bean，
    不管是由@ComponentScan或@Bean注解声明的；

@Import
@SpringBootApplication
@Import(LoggerModuleConfiguration.class)
public class PartyApplication {}

3.2. @Enable*

  • 如同@EnableScheduling,我们需先定义@EnableLoggerModule注解；

@Retention(RetentionPolicy.RUNTIME)
@Target({ElementType.TYPE})
@Documented
@Import(LoggerModuleConfiguration.class)
@Configuration
public @interface EnableLoggerModule {}

  • 使用@Enable*注解

@SpringBootApplication
@EnableLoggerModule
public class PartyApplication {}

  • @EnableLoggerModule是对@Import的包装，
    包含多个配置的模块，把配置聚合到单模块更便捷；

3.3. Auto-Configuration

  • 自动加载模块来代替硬注入(hard-wiring)代码

  • classpath创建META-INF/spring.factories

org.springframework.boot.autoconfigure.EnableAutoConfiguration=\
  com.elf.logger.LoggerModuleConfiguration

4. 配置属性

4.1. Yaml配置

elf:
  uom:
    defaultCurrencyCode: CNY

4.2. 属性配置

  • @ConfigurationProperties

@ConfigurationProperties(prefix = "elf.uom")
public class UomProperties {

	private String defaultCurrencyCode = "CNY";

	public String getDefaultCurrencyCode() {
		return defaultCurrencyCode;
	}

	public void setDefaultCurrencyCode(String defaultCurrencyCode) {
		this.defaultCurrencyCode = defaultCurrencyCode;
	}
}

4.3. 启用属性配置

  • @EnableConfigurationProperties

@RestController
@RequestMapping("/uom")
@EnableConfigurationProperties({
	UomProperties.class
})
public class UomController{

	private UomProperties uomProperties;

	@Autowired
	public UomController(UomProperties uomProperties) {
		this.uomProperties = uomProperties;
	}

	@PostMapping("/getUomCode")
	public ResponseEntity<String> getUomCode() {
    	String currencyCode = uomProperties.getDefaultCurrencyCode();
    	return new ResponseEntity<>(currencyCode, HttpStatus.OK);
	}
}
