1. Comment 注释

<!-- comment -->

[//]: # (comment)

[//]: <> (comment)

[^_^]: # (comment)

[comment]: <> (comment)

<div style='display: none'>comment</div>
select * from party

2. List 列表

2.1. OL 有序列表

  • 数字加英文句号(.)

1. Pal
2. Elf
3. Lua

2.2. UL 无序列表

  • 星号(*) 加号(+) 减号(-) 三者选其一

  • 列表符号与内容之间需要一个英文空格

* Red
+ Green
- Blue

2.3. NL 列表嵌套

  • 上一级与下一级之间三个空格

+ Elf
+ Cpp
+ Java
   1. Thread
   2. IO
   3. Net

3. Checkbox 复选框

  • -和[间有一个空格

  • [和]间需一个空格

  • ]和后面文字间需一个空格

- [ ] hello
- [ ] world
- [ ] end

4. Table 表格

第二行分割表头和内容:-有一个就行,为对齐,可酌情增减;
文字默认居左,-两边加:文字居中,-右边加:表示文字居右;
注:原生语法两边都要用|包起来,此处省略

姓名|技能|排行
--|:--:|--:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟

5. Source Highlight

5.1. 单行代码

  • 代码左右分别用一个反引号包起来

`single line code`

5.2. 代码块

  • 代码前后分别用三个反引号抱起来,且两边的反引号单独占一行

```
System.exit(-1);
```

5.3. 原样显示

`<pre>`和`<code>`

6. Link 链接

6.1. Image 图片

![alt text](img url "Optional title")
![Security](https://ico.ooopic.com/ajax/iconpng/?id=186750.png "安全")

子路径
![avatar](folderName/imgName.jpg)

当前路径: ./
![avatar](./folderName/imgName.jpg)

父路径用"../":
![avatar](../imgName.jpg)

6.2. Hyper Link 超链接

(title是英文双引号,可选项):[超链接名](超链接地址 ''超链接title'')
[百度](http://baidu.com "百度")
<a href="超链接地址" target="_blank">超链接名</a>
<a href="http://baidu.com" target="_blank">百度</a>

6.3. Automatic Link 自动链接

  • 用尖括号包起来,处理网址和邮件地址

<http://www.baidu.com>
<address@email.com>

6.4. Anchor 锚点

[锚点说明](hashtag(#)anchorId);
<h1 id="anchorId">目标锚点</h1>

7. Format 格式化

7.1. Bold 加粗

  • 加粗文字左右分别用两个*号包起来

**加粗**

7.2. Italic 斜体

  • 要倾斜的文字左右分别用一个*号包起来

*斜体*

7.3. Italic & Bold 斜体加粗

  • 要倾斜和加粗的文字左右分别用三个*号包起来

***斜体与加粗***

7.4. Line-Through 删除线

  • 要加删除线的文字左右分别用两个~~号包起来

~~删除线~~

7.5. Discoloration 变色

$\color{red}{红色}$
<font color=silver>银灰色</font>
$\color{#0000FF}{蓝色}$

8. Block Element 区块元素

8.1. 类Setext形式

  • 用底线形式,利用=(最高阶标题)和-(第二阶标题),任何数量的=和-均可

High Level Title
================

Low Level Title
---------------

8.2. 类Atx形式

  • 在行首插入1~6个#,对应到标题1到6阶

# H1 Title
## H2 Title
### H3 Title
#### H4 Title
##### H5 Title
###### H6 Title

8.3. 选择性的闭合类Atx样式标题

  • 在行尾加#,而行尾#数量也不用和开头一样,

  • 行首的#字符数量决定标题的阶数

# H1 Title ######
## H2 Title #####
### H3 Title ####
#### H4 Title ###
##### H5 Title ##
###### H6 Title #

8.4. 区块引用Blockquote

  • 在每行最前面添加>,也可以只在第一行最前加>

> all features are available free of charge.
> It is a full version: no hidden payments,demo versions,or limitations.

> all features are available free of charge.
  It is a full version: no hidden payments,demo versions,or limitations.

8.5. 引用可嵌套

  • 输入n个>即代表引用n层

> This is the first level of quoting.
>
>> This is nested blockquote.
>
> Back to the first level.

原内容
>一层引用原内容
>>二层引用原内容
>>>三层引用原内容

8.6. 分隔线

  • 在一行中用三个以上的星号,减号,底线来建立一个分隔线,
    行内不能有其他东西,可以在星号或是减号中间插入空格

9. Other 其它

9.1. 换行

9.2. 特殊字符自动转换 <和&

  • <符号用于起始标签

  • &符号用于标记html实体

  • 必须要使用实体的形式,像是<和 &

「 AT&T」 => 「AT&amp;T」   4 < 5 => 4 &lt; 5
& => https://www.google.com/search?num=30&amp;q=elf&amp;tbm=isch
