1. Asciidoctor Plugin

1.1. Environment

  • Msys264,RubyInstaller,Asciidoctor

  • GRAGHVIZ_DOT ⇒ E:\Install\GraphvizV24\bin

  • PATH ⇒ %NODE_PATH%,%GRAGHVIZ_DOT%

1.2. Generic

gem install asciidoctor

asciidoctor -a toc=left fileName.adoc

1.3. Highlight

gem install rouge

gem install coderay

:source-highlighter: rouge

1.4. Diagram

gem install asciidoctor-diagram

asciidoctor -r asciidoctor-diagram fileName.adoc

1.5. Math

# https://github.com/asciidoctor/asciidoctor-mathematical

gem install asciidoctor-mathematical

asciidoctor -r asciidoctor-mathematical fileName.adoc

3. AsciiDoc Table

水平对齐

<

水平居左

>

水平居右

^

水平居中

垂直对齐

.<

垂直居左

.>

垂直居右

.^

垂直居中

跨行和跨列

.row+|

跨行

column+|

跨列

colomn.row+|

跨行和列

水平和垂直对齐

colomn.row+horizontal.vertical|

2.3+^.^：跨2列三行并水平和垂直居中
