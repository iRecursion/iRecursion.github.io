1. Antora Installation

  • 以下所有插件均在asciidoc-generator项目目录下执行安装

mkdir asciidoc-generator && cd asciidoc-generator

node -e "fs.writeFileSync('package.json', '{}')"

npm i -D -E @antora/cli@3.1 @antora/site-generator@3.1

# 找不到antora则在Node目录执行
npm i -g @antora/cli@3.1 @antora/site-generator@3.1

antora -v

2. Antora Operation

echo off
cls
chcp 65001

set BASE_URL=D:/Install/Dev/workspace/AsciiDoc/asciidoc-antora
set ANTORA_URL=%BASE_URL%/asciidoc-generator/
set SITE_URL=%BASE_URL%/asciidoc-distro/docs/index.html

d:
cd %ANTORA_URL%

antora --extension @antora/lunr-extension --fetch antora-playbook.yaml & start %SITE_URL%

3. Antora Github

  • 创建仓库username.github.io：iRecursion.github.io

  • 仓库根目录下创建/docs目录，用于存放antora生成的文件

  • 在/docs目录下创建.nojekyll，此文件可空内容，
    https://docs.antora.org/antora/latest/publish-to-github-pages/

  • 点击仓库 → Settings → Pages(左侧)
    → 可选择分支，文件夹，自定义域名，取消发布站点

  • https://iRecursion.github.io

4. MathJax

  • https://asciimath.org/#syntax

  • https://docs.mathjax.org/en/latest/

  • https://gitlab.com/djencks/asciidoctor-mathjax.js

  • asciidoctor-mathjax提供Asciidoctor.js扩展，
    使用Mathjax 3 JavaScript在服务器端呈现AsciiMath和LaTeX；

npm i @djencks/asciidoctor-mathjax
asciidoc:
  attributes:
    stem: 'asciimath'
  extensions:
  - '@djencks/asciidoctor-mathjax'
****
stem:[\vec{A} * \vec{B} = A_1B_1 + A_2B_2]
****
