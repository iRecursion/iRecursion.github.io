1. 八卦概述

1.1. 先天八卦

  • 源于河图，乾坤定南北，坎离定东西，天南地北为序，
    上为天乾，下为地坤，左为东离，右为西坎。

  • 先天八卦数：乾一、兑二、离三、震四、巽五、坎六、艮七、坤八。

  • 中间数为0，以代表五或十，0象征宇宙元气。它的序数对宫相加之和为九数；

  • 先天八卦演变过程：从太极，两仪，四象，到最后八卦，此谓宇宙形成过程；

2. 后天八卦

  • 源于洛书。离坎定南北，震兑定东西。

  • 后天八卦数：坎一、坤二、震三、巽四、中五、乾六、兑七、艮八、离九。

  • 中间数为五，与对宫纵横相加之和为十五数。

3. 先后天八卦

  • 先天八卦：

    • 老父在南，老母在北，长男居东北，长女居西南。

    • 中男在正西，中女在正东，少男居西北，少女居东南。

  • 后天八卦：
    **老父在西北，老母在西南，长男居东方，长女居东南。

    • 中男居正北，中女居正南。少男居东北，少女居正西。

  • 应用：

    • 先天八卦取数，后天八卦取向；后天八卦应用多，先天八卦少；

    • 先天八卦数：乾一、兑二、离三、震四、巽五、坎六、艮七、坤八；

    • 后天八卦方向：震卦，正东；巽卦，东南；离卦，正南；
      坤卦，西南；兑卦，正西；乾卦，西北；坎卦，正北；艮卦，东北

4. 八卦代数与方位

  • 卦形歌诀：乾三连，坤六断，震仰盂，艮覆碗，离中虚，坎中满，兑上缺，巽下断。

  • 先天八卦：乾一，兑二，离三，震四，巽五，坎六，艮七，坤八。

  • 后天八卦：坎一、坤二、震三、巽四、五为中宫，乾六、兑七、艮八、离九。

  • 先天八卦：乾南，坤北，离东，坎西，兑东南，震东北，巽西南，艮西北。

  • 后天八卦：震东，兑西，离南，坎北，乾西北，坤西南，艮东北，巽东南。
